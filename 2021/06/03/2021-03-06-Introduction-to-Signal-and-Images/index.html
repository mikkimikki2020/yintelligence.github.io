Introduction to Signal and Images

Introduction to signals

Definition

The variation of phenomena as a function of one or multiple variables, often time

  • The potential of circuit as a function of time
  • DNA protein pairs as a function of their position （sequence）
  • A movie, which is a 2D signal as a function of time （or 3D signal）

In this course, we will mostly refer to

  • Signals often for 1D representations
  • Images for higher dimensions

Continuous signals

Often the （source of the） signal is inherently continuous

  • It has a value at any t

  • Notation s（t）, using a continuous argument t \[ s : R → R : s (t) = { t ∈ R : s (t) ∈ R } \]

Examples:

  • Analogue electrical potential measurements over time

  • In 2D: analogue films such as X -ray

Discrete signals

Sometimes the value is only available at discrete instances

  • Output of the heart, available at each heartbeat
  • DNA Sequences, the precise order of nucleotides in a DNA molecule

Or the value was measured at discrete （time） points

  • Also referred to as sampling
  • e.g. the weight of a person, as a function of age

Notation using index i instead of continuous argument \[ s : N → R : s [t] = { i ∈ N : s (i) ∈ R } \] Or （limited set of values） \[ s_i = s[i] = s(t_i)\ where\ i = 1,...,N \]

Discretization or sampling

Transform a continuous signal into a set of values at discrete time points

Occurs on the independent variable, t in our case \[ s : R → R : s (t) = { t ∈ R : s (t) ∈ R }\\s_i = s[i] = s(t_i)\ where\ i = 1,...,N \]

Only the discrete measurements are retained

  • If needed, values are interpolated at other time points

Sampling has to be done with care, to prevent loss of information

  • Sampling theorem of Shannon / Nyquist

Quantization

Transforms the amplitude of a signal, to a limited number of values

  • Values are quantified using a finite accuracy

Occurs on the dependent variable

  • The value or amplitude of the signal, here s（t）

Values are chosen that can be efficiently represented using computer arithmetic

  • char, integer, float, double

    8 bit → 28 = 256 levels

    16 bit → 216 = 65536 levels

    32 bit → 232 = 4294967296 levels

Size of a signal

An example of a typical ECG signal

  • Electrical measurements, roughly between -5mV, 5mV

    Analogue, continuous signal

  • After anti-aliasing filter, sampled

    Ideally 250Hz to 1 KHz （1000 values per second）

  • Quantization over a range of ±10 mV

    Amplitude representation using 12bit to 16bit （65536 levels）

  • 1 min of ECG （signal electrode）

    60 x 1000 x 2B = 117kB

Properties of Signals

Energy and power

  • Signal Energy and Signal Power \[ E_s = \int^{t_2}_{t_1}|s(t)|^2dt\\ P_s = \frac{1}{t_2-t_1}\int^{t_2}_{t_1}|s(t)|^2dt \]

  • Or for discrete signals \[ E_s = \sum^{n_2}_{i=n_1}|s[i]|^2\\ P_s = \frac{1}{n_2-n_1+1}\sum^{n_2}_{i=n_1}|s[i]|^2\\ \]

Periodicity

Some signals may have a repeating pattern \[ s(t + T) = s(t) \]

  • If this holds for T= Tp , then also for 2Tp, 3Tp, …
  • The smallest value of T is the fundamental period T0

In case of discrete signals we can state there is a number N for which \[ s[i + N] = s[i] \]

  • The smallest of such numbers being the fundamental period N0

Important signals

Sinusoidal signals

\[ s(t) = A\cos(\omega_0t + \phi) \]

Or discrete \[ s_n = s[n] = A\cos(2\pi f_0t_n + \phi) \]

  • t : the time in seconds （s）
  • A : the amplitude
  • ω0: the angular frequency （2πf） in Hz or radians / s
  • ɸ : the phase angle or offset in radians

Complex exponentials

\[ s(t) = A\cos(\omega t + \phi) + iA\sin(\omega t + \phi) \]

  • Real and imaginary part can be separated
  • Each correspond to sinusoids

Harmonically related exponentials \[ s(t) = Ae^{i(k\omega t + \phi)} \]

Properties of the delta function

  • Relation with the step function

    • The delta function is the first derivative of the unit step function \[ \delta (t) = \frac{\partial u(t)}{\partial t} \]

    • The step function is the running integral of the delta function \[ u(t) = \int^{t}_{-∞}\delta(\tau)d\tau \]

    integral in which independent variable （e.g. t） is part of the integration boundaries

  • Sifting property

    • Consider a delayed pulse at n=n0 \[ \delta [n-n_0] \]

    • Multiply it with a signal x[n] \[ \sum^{+∞}_{n=-∞}x[n]\delta [n - n_0]=x[n_0] \]

Sift out the value at n0

Signals as linear combination of impulses

We can write a signal x[n] as

  • Superposition of shifted and scaled impulses

\[ x[n] = ... + x[-2]\delta [n+2] + x[-1]\delta [n+1] + x[0]\delta [n] + x[1]\delta [n-1] + x[2]\delta [n-2]\\ with\ \left\{ \begin{aligned} \delta [n+k]=1 &, & n = -k \\ \delta [n+k]=0 &, & n ≠ -k \end{aligned} \right. \]

Write a discrete signal as

  • Linear combination of weighted, shifted impulses

\[ x[n] = \sum^{+∞}_{k=-∞}x[k]\delta [n - k] \]

The rectangular function
  • Squared pulse

\[ p(t)=\left\{ \begin{aligned} 0 & , & |t| ＜ \frac{1}{2} \\ \frac{1}{2}& , & |t| = \frac{1}{2} \\ 1 & , & |t| ＞ \frac{1}{2} \end{aligned} \right. \]

The sinc function
  • Normalized sinc funtion when using πt

\[ sinc[t]=\left\{ \begin{aligned} \frac{\sin(\pi t)}{\pi t} & , & t ≠ 0 \\ 1 & , & t= 0 \end{aligned} \right. \]

Basic Operations on Signals

  • Time shifting

    • A time shifted version of the signal
    • x [n] → x [n-n0]

  • Time reversal

    • Reverses the time axis
    • x [n] → x [-n]

  • Sum and scaling of signals

    • Sum / scale of corresponding values

  • Autocorrelation \[ R_{xx}(\tau) = \int^{+∞}_{-∞}x^{*}(t)x(t+\tau)dt \]

    • New signal as a function of delay
    • Measures correspondence with previous values
    • An energy function Rxx（0）= Es

  • Convolution

    • Mathematical operation between two signals, gives new signal \[ (f*g)(\tau) =\int^{+∞}_{-∞} f(t)g(\tau-t)dt,\\ (f*g)(\tau) =\sum^{+∞}_{m=-∞} f[m]g(\tau-n) \]

    • Running integral of a time reversed signal with another signal

    • Convolution is associative, commutative and distributive

Linear systems

A system is a mathematical description of a process

  • Or approximation of a physical process
  • Continuous, discrete or mixed systems

Takes a signal on input, processes it and yields an output

  • Transforms one signal into another signal

Transformation between x（t） and y（t） given by:

  • Differential equation （continuous system）
  • A difference equation （discrete system）

Some properties of systems

Invertible systems

A system is said to invertible if there exists a system such that \[ w[n] = Wy[n] = Wℒx[n] = x[n] \]

A memory-less system

If the output at any time is only dependent on the input at that same time

Stable system

If a finite input signal always generates a finite output signal

Examples

  • An amplifier

    Invertible? Memory-less? Stable? \[ y[n]=3x[n] \]

  • A moving average

    Invertible? Memory-less? Stable? \[ y[n]=\frac{1}{2}(x[n]+x[n-1]) \]

Linear vs non-linear systems

  • Additivity

    • The response of the superposition of two input signals is the superposition of the responses \[ ℒ(x_1[n] + x_2[n]) = ℒx_1[n] + ℒx_2[n] = y_1[n] + y_2[n] \]

  • Homogeneity

    • The response of a scaled version of the input signal is scaling of response \[ ℒ(ax[n]) = aℒx[n] = ay[n] \]

  • Linearity

    • A system is linear if a linear combination of the input signals is transformed into a linear combination of the separate responses （= additivity + homogeneity） \[ ℒ(ax_1[n] + bx_2[n]) = aℒx_1[n] + bℒx_2[n] = ay_1[n] + by_2[n] \]

Time-variant vs time-invariant systems

Time-invariant or Time-shift-invariant

  • If a shift of the input signal causes the same shift in the output signal \[ ℒx[n-n_0] = y[n-n_0] \]

  • Examples \[ y[n] = \sum^{n}_{k=-∞}x[k]\\ y[n] = \sin(x[n]) \]

Time-varying

  • The output value cannot be determined without knowing the current time

  • A shift of the input signal will not cause the same shift in the output signal \[ y[n] = nx[n] \]

Causal vs Non-causal systems

Causal system

  • When the output at a certain moment only depends on inputs from that and earlier moments

    • Not on input at later moments
    • All physical systems are causal

  • Example \[ y[n] = x[n] - 2x[n-1] + x[n-2], \]

Non-causal systems

  • Output depends on inputs at a later time, future inputs \[ y[n] = x[n + 1] \]

Systems that are linear and time-invariant

Remember: we can write a signal x[n] as Linear combination of weighted, shifted impulses \[ x[n] = \sum^{+∞}_{k=-∞} x[k]\delta [n-k]\ with\ \left\{ \begin{aligned} \delta [n+k]=1 &, & n = -k \\ \delta [n+k]=0 &, & n ≠ -k \end{aligned} \right. \] So if we know the response to an impulse…. We can predict the response to any input

Only valid for linear, time-invariant systems

\[ h[n] = ℒ[n]\\ y[n] = ℒx[n]=\sum^{+∞}_{k=-∞} x[k]h[n-k] \]

Linear Time Invariant （LTI） Systems

The powerful property of discrete LTI- systems

  • Consider a discrete, causal LTI system ℒ.

    The response to an impulse is h（n） \[ h[n] = ℒ\delta[n] \]

  • Any input can be represented as a linear combination of basic signals （e.g. impulses） \[ x[n] = \sum^{+∞}_{k=-∞}x[k]\delta [n - k] \]

  • The response of an LTI system is the superposition of responses to the basic signals \[ y[n] = ℒx[n] \]

  • A discrete LTI system is fully characterized through the response to an impulse \[ ℒ\sum^{+∞}_{k=-∞}x[k]\delta [n - k] =\sum^{+∞}_{k=-∞}x[k]h[n - k] \]

Impulse response to characterize a LTI system

  • LTI system \[ y[n] + 0.4 y[n-1] = 3x[n] \]

  • Solve for x[n] = δ[n], assume y[-1]=0 \[ y[0] = 3x[0] - 0.4y[-1] = 3\\ y[1] = 0 - 0.4(3) = -1.2\\ y[2] = 0 – 0.4(-1.2) = 0.48\\ y[n] = (-0.4)^n(3)\\ h[n] = 3 (-0.4)^n u[n]\\ \]

System response as a convolution

  • A discrete LTI system \[ y[n] = ℒx[n]\ with\ h[n] = ℒδ[n],\ ℒδ[n-k] = h[n-k] \]

  • An arbitrary input \[ x[n] = \sum^{+∞}_{k=-∞}x[k]\delta [n - k]\\ℒ\sum^{+∞}_{k=-∞}x[k]\delta [n - k] =\sum^{+∞}_{k=-∞}x[k]h[n - k] \]

  • The system response can be found as the convolution of the input with the impulse response \[ y[n]=\sum^{+∞}_{i=-∞}x[i]h[n - i]\\ y[n]=\sum^{+∞}_{i=-∞}x[n-i]h[i]\\ y[n] = x[n]* h[n] \]

Frequencies and Fourier series

Frequencies

  • For cyclical processes

    • The frequency is the number of cycles, or periods per unit of time: f = 1 / T

      Expressed in Herz, cycles per s （or per mm）

  • Examples

    • Electricity （AC） at 50 Hz – Human hearing between 50Hz and 20kHz

Frequencies and the frequency domain

  • Why go to the frequency domain?

    • LTI systems can be fully described using impulse response

    • An alternative is the frequency response

      Frequency or Spectral Analysis

      Bode diagram

  • Advantages

    • Some mathematical techniques are easier in frequency domain
      • Convolution → product
      • Differential equation → algebraic equation
    • Some applications are more easily described or characterized in frequency domain
    • Measurement tools exist to measure frequency response

Fourier series

Periodic signals can be decomposed

  • As a sum of sinusoidals, i.e. Fourier series,
  • In different frequencies

Every periodic function x（t） with period T can （exactly） be represented as a weighted sum of sinusoidals given that the following conditions are correct （Dirichlet conditions）

  • x（t） must be absolutely integrable in the interval [0, T]
  • x（t）has a finite number of maxima and minima in the interval [0, T]
  • x（t） has a finite number of discontinuities in the interval [0, T]

Reminder: Important signals - Complex exponentials

Any periodic signal can be represented as a sum of sinusoids \[ x(t) = \sum^{k=-∞}_{+∞}a_ke^{jk\omega_0t}\\ x(t) = \sum^{+∞}_{k=-∞}a_k\cos(k\omega_0t)+ja_k\sin(k\omega_0t) \] The coefficients ak can be found \[ a_n =\frac{1}{T}\int^{T}_{0}x(t)e^{-jn\omega_0tdt} \]

Fourier Transform

Fourier Series

  • Each periodic signal can be represented as a sum of sinoids

Fourier Transform

  • Extension to aperiodic signals
  • Aperiodic signals are periodic, with infinite period

Also a Discrete Fourier transform

  • With a fast algorithm （FFT）

\[ x(t) = \frac{1}{2\pi}\int^{+∞}_{-∞}X(\omega)e^{j\omega t}d\omega\\ X(\omega) =\int^{+∞}_{-∞}X(t)e^{j\omega t}dt\\ H_n=\sum^{N-1}_{k=0}h_ke^{\frac{2\pi ikn}{N}}\\ h_k=\frac{1}{N}\sum^{N-1}_{k=0}H_ne^{\frac{-2\pi ikn}{N}} \]

Discrete Fourier Transform （DFT）

The DFT provides uniformly spaced samples of the Discrete-Time Fourier Transform （DTFT） \[ X[k] = \sum^{N-1}_{n=0}X[n]e^{-\frac{2\pi jkn}{N}}\\ X[n] = \frac{1}{N}\sum^{N-1}_{k=0}X[k]e^{\frac{2\pi jkn}{N}} \] Requires N2 complex multiplications and N（N-1） complex additions

  • Computationally expensive
Faster DFT computation? FFT

Take advantage of the symmetry and periodicity of the complex exponential

let \[ W_N=e^{\frac{-j2p}{N}} \]

  • symmetry \[ W^{k[N-n]}_{N}=W^{-kn}_{N}=(W^{kn}_{N})^{*} \]

  • periodicity \[ W^{kn}_{N}=W^{k[n+N]}_{N}=W^{[k+N]n}_{N} \]

  • Note that two length N/2 DFTs take less computation than one length N DFT: \[ 2（\frac{N}{2}）^2 ＜N^2 \]

  • Algorithms that exploit computational savings are collectively called Fast Fourier Transforms

Properties of （Discrete） Fourier Transform

Important signals and Fourier transforms

Fourier series of LTI systems

  • Complex exponential on the input \[ x[n] = e^{j\omega t_n} \]

    • Functions of this type are eigenfunctions of the LTI system

  • The system response can be written as a convolution

  • The output is a complex exponential weighted by a complex gain factor H（⍵）

    • H（⍵） is an eigenvalue and is dependent on frequency of the input signal x[n]

  • For a linear combination of complex exponentials \[ x[n]=\sum_kc_ke^{j\omega_kt_n}\\ y[n]=\sum_kc_kH(\omega_k)e^{j\omega_kt_n} \]

    • If the input can be represented as a linear combination of complex exponentials
      • Using Fourier transform, any signal can be written in this form
    • Then the output can also be represented with a complex exponentials
      • Each linear coefficient is scaled according to the frequency
      • The output of the filter depends on the frequency

Bode diagram for characterizing an LTI-system

  • Compute or measure H（⍵） for all frequencies

  • Describe H（⍵） in Bode diagram

    Frequency on a logaritmic scale

    • Amplitude-Bode diagram
      • Amplification|H（⍵）| expressed in dB （logarithmic）
    • Phase-Bode diagram
      • Phase shift Arg（H（⍵）） in degrees or radians

\[ |H(\omega)|=\sqrt{R(H(\omega))^2+T(H(\omega))^2}\\ ∠(H(\omega))=arctan\frac{R(H(\omega))}{T(H(\omega))} \]

Filters

  • Every sinusoidal component is modified
    • Amplitude and phase multiplied with the corresponding value of H（f）
  • When we eliminate or supress frequencies, the LTI-system is called a frequency-band filter
    • Named by how it changes the frequency spectrum

Filtering in frequency domain

Principle

  • Input signal in time domain
    • The original time-series
  • Convert signal to frequency domain
    • Fourier transform
  • Filter / select the frequencies you want to keep
    • Multiply with a rectangular pulse
  • Transform back to time domain
    • Inverse Fourier transform
  • The output is a low / band / high-pass filtered signal

Implementation of digital filters in the temporal domain

  • Finite impulse response filters （FIR）

    • h[k] is finite, or h[k] is zero for k larger than a certain value K

    • General form: \[ y[n]=\sum^{M}_{k=-N}b_kx[n-k] \]

  • Infinite impulse response （IIR）

    • Output depends on x[k] and y[k] （feedback）

Sampling Theorem

When we sample a continuous signal information can get lost, for certain conditions it is possible to reconstruct the exact continuous signal back from the discrete sampled values

Sampling or Nyquist theorem

  1. x（t） has to be bandlimited \[ X(f)=0\ for\ all\ |f| ＞ B \]

  2. The sampling frequency has to be larger than twice the highest frequency in the signal

If Nyquist is not fulfilled: Aliasing

Summary

Signals

  • Variation of a physical measure as function of a variable
    • Often as a function of time
  • Digital signals obtained through discretisation and quantization
    • Sampled at certain time points, represented with finite accuracy
  • Important signals
    • Sinusoids, step, delta, pulse and sinc function
  • Important functions
    • Time shifting and reversal, autocorrelation, convolution

Systems

  • Mathematical description or approximation of a process
    • Gives an output given an input
  • Linear time-invariant systems （LTI）
    • Linear systems that don’t change over time
  • Impulse response to characterize an LTI system
    • A signal is the sum of scaled shifted impulses
    • The output is the sum of scaled, shifted impulse responses
  • System response of a system
    • Convolution of the input with the impulse response

Frequencies and Fourier analysis

  • An alternative way to describe the response of a system
    • Some applications are more easily described in frequency domain
  • Fourier transform （Fourier analysis）
    • Decompose a signal in its frequency components
  • Bode diagram to characterise an LTI system
    • If we know the response for all frequencies
    • We can describe the output to any input
  • Common systems used as a filter
    • Low-pass, high-pass, band-pass or band-stop

Introduction to images

Definition

Images: Any （2D, 3D, 4D, …） representation, taken by a camera, telescope, microscope,… or medical imaging device

Medical images:

  • Usually 2D slices or projections, 3D Volumes

    • With pixels or voxel elements

  • But also 2D+T and 3D+T

    • Dynamic sequences, etc

Discrete Images

Mathematically, images can be seen as N-Dimensional intensity functions of the spatial coordinates \[ f(x, y) = i \] In practice, we use discrete images

  1. Discretization \[ f(m, n)\ with\ m = 0, 1, 2,...,M-1; n = 0, 1,...,N 1 \]

  2. Quantization

    • Integer or floating point representation using 8, 12 or 16-bit representation

Grey scale and color images

Images need to be visualized

  • This requires a relation between
    • The image intensity,
      • A physical measurement
    • The displayed color or grey scale
  • Using a look-up table
    • The table does not affect the image （stored on disk）
    • Determines visualisation （shown on screen）

Color map

  • grey scale
  • full color Lookup table （LUT）

Images as a measurement in physical space

  • Medical images are usually “calibrated”

    • Dimensions of a pixel known with great certainty
    • Position of image known with respect to room

  • Images are used for treatment planning and guidance

  • Spatial information

    Stored in the medical image header, referencing the image w.r.t. the room or scanner

    • Origin
      • The physical coordinate of the first pixel
    • Spacing
      • The physical space between pixels in each dimension

The size of an image

A example of a typical CT sequence

  • A slice stored in signed short （2B）
    • 512 x 512 x 2B = 512kB + header
  • A series of 200 slices （e.g. 200mm）
    • 200 x 512kB = 100MB （uncompressed）
  • Lossless compression （12bit+other redundancy）
    • ~50MB （depends on image content）
  • 30 volumes of a dynamic sequence
    • 30 x 50MB = 1.5GB

Fourier Transform

2D Fourier Transform

  • Extension to 2D of Fourier transform

    • Horizontal frequency u, vertical frequency v \[ T(u, v)=\int^{+∞}_{-∞}\int^{+∞}_{-∞}f(x,y)e^{-j2\pi(ux+vy)}dxdy \]

  • Inverse Fourier transform \[ f(x,y)=\int^{+∞}_{-∞}\int^{+∞}_{-∞}T(u, v)e^{j2\pi(xu+yv)}dudv \]

2D Discrete Fourier Transform

  • The 2D DFT is separable and symmetric

\[ F(k,l)=\frac{1}{MN}\sum^{M-1}_{m=0}\sum^{N-1}_{n=0}f(m,n)e^{-j2\pi (\frac{km}{M}+\frac{nl}{N})} \]

  • The inverse DFT （IDFT） \[ f(m,n)=\sum^{M-1}_{m=0}\sum^{N-1}_{n=0}F(k,l)e^{j2\pi (\frac{km}{M}+\frac{nl}{N})} \]

Properties of 2D （Discrete） Fourier Transform

  • Energy-conserving （Parseval’s theorem） \[ \int^{+∞}_{x=-∞}\int^{+∞}_{x=-∞}|f(x,y)|^2dxdy=\int^{+∞}_{u=-∞}\int^{+∞}_{u=-∞}|F(u,v)|^2dudv \]

  • Linear transformation \[ h(x, y) = \alpha f(x, y) + \beta g(x, y) ⇔ H(u, v) = \alpha F(u, v) +\beta G(u, v) \]

  • Periodic signal \[ F(k,l) = F(k ± \alpha M,l ± \beta N)\ where\ \alpha,\beta\ are\ integers \]

  • Conjugate symmetric for real images \[ F(-k, -l) = F^{*}(k,l) → |F(-k, -l)| = |F^{*}(k,l)|\ and\ ∠F(k, l) = -∠F(k,l) \]

A spatial shift or translation leads to an additional linear phase component: \[ f(m-m_0, n-n_0) ⇔ F(k,l)\exp(-j2\pi(\frac{km0}{M} +\frac{\ln0}{N})) \] Rotation of an image leads to a rotation of the Fourier Transform

Scaling an image leads to an inverse scaling of its Fourier transform \[ f(am,bm)⇔ \frac{1}{|ab|}F(\frac{k}{a},\frac{l}{b}) \]

2D Convolution

  • Extension of convolution to 2D and discrete cases \[ f(x,y)*g(x,y)=\int^{+∞}_{-∞}\int^{+∞}_{-∞}f(\alpha,\beta)g(x-\alpha,y-\beta)d\alpha d\beta \\ f(x,y)*g(x,y)=\sum^{K-[\frac{K}{2}]-1}_{K-[\frac{K}{2}]}\sum^{L-[\frac{L}{2}]-1}_{L-[\frac{L}{2}]}f(k,l)g(x-k,y-l) \]

  • Convolution leads to multiplication in spectral domain, and vice versa \[ h(x,y)*f(x,y) ⇔ H(u,v).F(u,v)\\ h(x,y).f(x,y) ⇔ H(u,v)*F(u,v) \]
