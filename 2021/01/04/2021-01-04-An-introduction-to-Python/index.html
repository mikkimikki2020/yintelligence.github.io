An Introduction to Python

Python as a smart calculator

7 arithmetic operators

x = 5
y = 6
z = x * y

print(x, y, z)

Output: 5 6 30

  • x, y and z are variables
  • x is assigned the value 5
  • y is assigned the value 6
  • z is assigned the value of the multiplications of the values of x and y

We assume this is Python 3.X. In Python 2.X the last line would be print x, y, z

+ Sum
- Subtraction
* Multiplication
/ Division
% Modulus
** Exponent
// Integer division

Warning In Python 2.X / is the integer division

Operator precedence

Precedence Operator Operation
Highest ** Exponentiation
- Negation
*, /, //, % Multiplication, division, etc
Lowest +, - Addition and subtraction

More operations

import math

x = math.cos(math.pi)
y = math.cos(math.pi/2)

print(x,y)

Output: -1.0 6.12323399574e-17

  • Additional math operations can be found in the math module
  • Use the import keyword to load a module
  • All operations from a module must have the right prefix

Or alternatively:

from math import *

x = cos(pi)
y = cos(pi/2)
print(x,y)

Don’t forget to write the following line at the top of your Python program:

from math import *

Functions

Computing something real \[ c = \sqrt{a^2+b^2-2ab\cos\theta} \]

from math import *

a = 3.0
b = 4.0
theta = pi/3
c1 = sqrt(a*a + b*b - 2*a*b*cos(theta))
c2 = sqrt(a**2 + b**2 – 2*a*b*cos(theta))

print(c1)
print(c2)

Note that, due to the precedence rules of arithmetic operators this code produces the correct result!

But… are we going to write the same snippet of code for different triangles?

from math import *

def computeside(a, b, theta):
	c = sqrt(a*a + b*b - 2*a*b*cos(theta))
	return c

result1 = computeside(3, 4, pi/2)
result2 = computeside(2, 3, pi/5)

print(result1, result2)
  • The concept of a function is similar to the mathematical concept.
  • Indentation matters! The tab defines what is part of the function and what is not

Function definition

def <<function_name>>(<<parameters>>):
	<<block>>
  • parameters is a comma separated list of variables
  • block is the function body and must contain at least one statement

Most function definitions will contain a return statement （optional）:

return <<expression>>

An example:

def convert_to_celsius(fahrenheit):
	return (fahrenheit - 32) * 5 / 9

Function call

<<function_name>>(<<arguments>>)

Rules for executing a function call:

  1. Evaluate each argument from left to right
  2. Pass resulting values into function
  3. Execute the function

Example:

def convert_to_celsius(fahrenheit):
	return (fahrenheit - 32) * 5 / 9
	
temp = convert_to_celsius(2*16)
print(temp)

Complete example

Function calls can be nested

def diff(x, y):
	return x - y

def prod(x, y):
	return x * y

def convert_to_celsius(temp):
	return prod(diff(temp,32), 5/9)

celsius = convert_to_celsius(32)
print(celsius)

What NOT to do…

def convert_to_celsius(fahrenheit):
return (fahrenheit - 32) * 5 / 9

temp = convert_to_celsius(2*16)
print(temp)

>> IndentationError: expected an indented block

Indentations are important!

def convert_to_celsius(fahrenheit):
	return (fahrenheit - 32) * 5 / 9
	
temp = convert_to_celsius(2*16,15)
print(temp)

Some examples

Making choices

if statement

Executes a block of statements only if a certain condition is met (i.e. True), otherwise the statements are skipped.

Syntax:

if condition:
	<<block>>

Example:

if grade >= 10:
	print(”Congratulations, you passed the course!”)

if/else statement

Executes a block of statements only if a certain condition is True, and another block of statements if the condition is False

Syntax:

if condition:
	<<block #1>>
else:
	<<block #2>>

Example:

if ph < 7.0:
	print(ph, ”is acidic”)
else:
	print(ph, ”is basic”)

Multiple conditions can be chained

Syntax:

if condition1:
	<<block #1>>
elif condition2:
	<<block #2>>
else:
	<<block #3>>

an example:

def price(age):
    if(age < 12):
    	return 5
    if(age < 18):
    	return 8
    if(age > 65):
    	return 9
    else:
    	return 10
    
print(price(5))
print(price(9))
print(price(31))

Repeating code

the for loop

Repeats a block of statements over a group of values

Syntax:

for <<variable>> in <<group of values>>:
	<<block>>

Example:

for x in range(1,5):
	print(x,” squared is ”, x**2)

Output:

1 squared is 1
2 squared is 4
3 squared is 9
4 squared is 16

The range function

The range function generates a sequence of integers.

Syntax:

range(stop)
range(start, stop)
range(start, stop, step)

Example:

print(”Countdown:”)
for x in range(5, 0, -1):
	print(x)

Output:

Countdown:
5
4
3
2
1

range（start,stop）produces a list of integers between start （included） and stop （excluded）

Class exercise

Write a function that sums all numbers from a to b （using a for loop）

def rangesum(a,b):
	for x in range(a,b):
		sum = sum + x
        
print(rangesum(3,6))

Output:

UnboundLocalError: local variable 'sum' referenced before assignment
def rangesum(a,b):
	sum = 0
	for x in range(a,b):
		sum = sum + x
		
print(rangesum(3,6))

Output:

None

Do not forget to return your result to the main program!

def rangesum(a,b):
    sum = 0
    for x in range(a,b):
        sum = sum + x
    return sum

print(rangesum(3,6))

Output: 12 WRONG

def rangesum(a,b):

print(rangesum(3,6))

Output: 18

def sumStartToEnd(start,end):
    sum = 0
    for n in range(start,end+1,1):
        sum = sum + n
    return sum

print(sumStartToEnd(a,b))

Write a function that computes the following sum up to n

\[ \sum^n_{i=0}\frac{1}{2^i} \]

def sumUp(n):
    sum = 0
    for i in range(n+1):
        k = 1/2**i
        sum = sum + k
    return sum
