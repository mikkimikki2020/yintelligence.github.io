Slide + Exercise

The data types we have seen so far:

Type Notation Examples
Integers int 1, 5, -10000, …
Floating point float -1.1, math.pi, …
Boolean bool True, False

But what if we want to work with many values? e.g. the number of thousands of litres of hot sauce consumed in the United States, Canada and Britain in the period 2007-2012

Country 2007 2008 2009 2010 2011 2012
Canada 12 18 13 22 19 18
United States 57 64 66 72 74 69
Britain 20 16 15 18 21 19

We would need 18 variables! And as the observations increase, so does the amount of variables

But with lists:

canada = [12, 18, 13, 22, 19, 18]
us = [57, 64, 66, 72, 74, 69]
britain = [20, 16, 15, 18, 21, 19]

Lists

General form:

listname = [<<expression1>>, …, <<expressionN>>]

Example:

mylist = [4, 12, 2, 34, 17]

first = mylist[0]
second = mylist[1]
print(first, second)

mylist[0] = 11
print(mylist)

Output:

4 12
[11, 12, 2, 34, 17]
first = mylist[0]
second = mylist[1]

Conceptually:

Adding elements to Lists

Example:

mylist = [4, 12, 2, 34, 17]

mylist.append(50)
mylist.append(18)
mylist.append(64)

print(mylist)

Output:

[4, 12, 2, 34, 17, 50, 18, 64]

Appending two Lists

Example:

pyListA = [34, 12]
pyListB = [4, 6, 31, 9]

pyListA = pyListA + pyListB

print(pyListA)

Output:

[34, 12, 4, 6, 31, 9]

Removing an element from a list

Example:

mylist = [1, 2, 3, 4]
print(len(mylist))

del mylist[1]
print(mylist)

Output:

4
[1, 3, 4]

Quick and easy but mind the readability of your code

Access and replace ranges

List slicing

mylist = [1, 2, 3, 4]
z = mylist[1:2]
mylist[1:2] = [55, 66]

print(z)
print(mylist)

Output:

[2]
[1, 55, 66, 3, 4]

Slices make a copy!

  • [a: b] means from a to b, without b
  • [0: b] can be written as [: b]
  • [a: end] can be written as [a :]
  • [0: end] is [:]

Slicing lists to remove ranges

mylist = [1, 2, 3, 4]
z = mylist[1:2]
mylist[1:2] = []

print(z)
print(mylist)

Output:

[2]
[1, 3, 4]

More on slicing

from start, to stop, every step element

seq = L[start:stop:step]

get every other item, starting with the first

seq = L[::2]

get every other item, starting with the second

seq = L[1::2]

Inserting and removing an element

pyList = [4, 12, 2, 34, 17, 50, 18, 64, 6]
pyList.insert(3,79)
print(pyList)
pyList.remove(4)

Output:

[4, 12, 2, 79, 34, 17, 50, 18, 64, 6]

removing 4

pyList = [4, 12, 2, 79, 34, 17, 50, 18, 64, 6]
pyList.remove(4)

List operations according to docs.python.org

  • list.append(x)

    Add an item to the end of the list; equivalent to a[len(a):] = [x].

  • list.extend(L)

    Extend the list by appending all the items in the given list; equivalent to a[len(a):] = L.

  • list.insert(i,x)

    Insert an item at a given position. The first argument is the index of the element before which to insert, so a.insert(0, x) inserts at the front of the list, and a.insert(len(a), x) is equivalent to a.append (x).

  • list.remove(x)

    Remove the first item from the list whose value is x. Gives an error if there is no such item.

  • list.pop([i])

    Remove the item at the given position in the list, and return it. If no index is specified, a.pop() removes and returns the last item in the list.

  • list.index(x)

    Return the index in the list of the first item whose value is x. It is an error if there is no such item.

  • list.count(x)

    Return the number of times x appears in the list.

  • list.sort()

    Sort the items of the list, in place.

  • list.reverse()

    Reverse the elements of the list, in place.

Looping over elements

for x in [16, 22, 3, 14]:
	print(x)

Output:

16
22
3

A bit more on Lists

An empty list:

mylist = []

A list can store objects of different types:

mylist = [’Albert’, ’Einstein’, 1879, 1955]

And yes, even other lists:

life = [[’Canada’, 76.5], [’United States’, 75.5], [’Mexico’, 72.0]]
print(life[2][1])

Output:

72.0

Nested lists (list of lists)

# Create an empty list.
elements = []

# Append empty lists in first two indexes.
elements.append([])
elements.append([])

# Add elements to empty lists.
elements[0].append(1)
elements[0].append(2)
elements[1].append(3)
elements[1].append(4)

# Display top-left element.
print(elements[0][0])

# Display entire list.
print(elements)
# Loop over rows.
for row in elements:
    # Loop over columns.
    for column in row:
    	print(column, end="")
    print(end="\n")
