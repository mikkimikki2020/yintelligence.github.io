Linked List in Java with Example

CS61B Discussion 3: Linked Lists & Arrays

链表linked list是一种简单的数据结构，它以线性linear方式存储数据，类似于向量Vector。

然而，虽然Vector具有固定长度fixed length，但linked list可以动态dynamically增长grow和收缩shrink。

这样做的结果是实现稍微复杂一些。动态长度是通过在元素链中存储数据来获得的。每个元素都连接到下一个元素。通过连接更多元素，链条可以增长。这允许进一步的灵活性：可以从链的中间删除一个元素（不复制数据），或者在链的中间添加一个新元素。

  • 顶部：表示为一串数字的linked list。
  • 底部：与list元素sequence相同的linked list。

上图给出了一个说明这个比喻的例子。顶行将linked list显示为一串（或链）数字。为了实现这一点，我们将稍微修改这个比喻（参见图的底行）：我们将构建一个列表元素链，其中每个列表元素连接到下一个列表元素。

请注意，在我们将在这里开发的实现中，链中的第一个元素是列表的第一个元素。链的前端没有“特殊”元素。最后一个元素没有连接到任何下一个元素，所以这个元素将连接到一个空指针null pointer。

这个linked list类是许多其他更复杂的数据结构的基础，包括双链表、循环列表、图……

Basic operations on a linked list

为了实现链表，构建了一个链表元素链train/chain of list elements。 列表元素由它自己的类class表示，只是一对两个元素，其中第一个元素存储数据，第二个元素包含指向下一个列表元素的指针。 因此，列表元素类的功能非常有限。

有一个构造函数可以生成一对给定的数据元素和一个给定的下一个指针，并且有一些方法可以获取和设置该对的第一部分和第二部分。

给定列表元素类，链表类用于存储指向第一个列表元素的pointer。 该pointer将存储在变量“head”中。 在列表类的构造函数中，列表元素“head”被初始化为 null，因为列表中还不包含任何元素。

	// This method returns whether the LinkedList is empty
	public boolean isEmpty() { 							//O(1)
		return size() == 0;
	}/* This class represents a list where each element has a pointer to the next.
 * Each element of the list is an instance of ListElement which contains two things.
 * el1, the data of itself and el2, a pointer to the next element of the list. 
 * There is one root 'head' and a counter 'count' for keeping track of the length.
 * Adding to or removing from the front (addFirst and removeFirst) operate in O(1)
 * while adding to and removing from the back (addLast and removeLast) operate in O(n).
 */

public class LinkedList {
	private class ListElement {
		private Object el1; //type
		private ListElement el2;

		public ListElement(Object el, ListElement nextElement) {
			el1 = el;
			el2 = nextElement;
		}

		public ListElement(Object el) {
			this(el, null); 
		}

		public Object first() {
			return el1;
		}

		public ListElement rest() {
			return el2;
		}

		public void setFirst(Object value) {
			el1 = value;
		}

		public void setRest(ListElement nextList) {
			el2 = nextList;
		}
	}
    
	private ListElement head;
    private int count;
    
	public LinkedList() {
		head = null;
        count = 0;
	}
	
    // This method returns the current size of the LinkedList
	public int size() {	//O(1)
		return count;
	}
    
	// This method returns whether the LinkedList is empty
	public boolean isEmpty() { //O(1)
		return size() == 0;
	}    
    
	// This method adds an element to the front of the LinkedList
    public void addFirst(Comparable o) { //O(1)
        head = new ListElement(o, head);
		count++;
	}
    
	// This method removes the first element of the LinkedList
	public void removeFirst() { //O(1)
		head = head.el2;
		count--;
	}    
    
    // This method returns the first element of the LinkedList
	public Object getFirst() { //O(1)
		return head.first();
	}

	// This method adds an element to the back of the LinkedList
	public void addLast(Comparable obj) { //O(n)
        ListElement last = new ListElement(obj, null);
		ListElement temp = head;
		if (temp != null) {
			while (temp.rest() != null) {
				temp = temp.rest();
			}
			temp.el2 = last;
			count++;
		} 
		else { // If the LinkedList is empty temp.el2 will be non-existent
            this.addFirst(obj);
		}
	} 
    
	// This method can be used to insert a certain element at a certain position in the LinkedList
	public void insert(int index, Comparable o) { //O(n)
		if (head == null) head = new ListElement(o, null);
		else {
			ListElement temp = head;
			while (0 < index) {
				temp = temp.rest();
				index--;
			}
			ListElement newElement = new ListElement(o, temp.rest());
			temp.setRest(newElement);
		}
	}
    
	// This method removes the last element of the LinkedList
	public void removeLast() {//O(n)
		ListElement temp = head;
		ListElement previous = null;
		if (temp != null) {
			if (temp.rest() == null) {
				head.el1 = null;
			} 
			else {
				while (temp.rest() != null) {
					previous = temp;
					temp = temp.rest();
				}
				previous.el2 = null;
			}
			count--;
		} 
		else {
			System.out.println("The list was already empty");
		}
	}
    
    
    // This method sets the nth element to a certain object
	public void set(int n, Comparable obj) { //O(n)
		ListElement temp = head;
		try {
			while (0 <= n) {
				if (n == 0) {
					temp.el1 = obj;
					return;
				}
				temp = temp.rest();
				n--;
			}
		} catch (NullPointerException e) {
			System.out.println("The list has less than n elements");
		}
	}

    // This method returns the last element in the LinkedList
    public Comparable getLast() { //O(n)
        ListElement temp = head;
        if (temp != null) {
            while (temp.rest() != null) {
                temp = temp.rest();
            }
            return temp.first();
        } 
        else {
            return "The list is empty";
		}
	}
    
    // This method checks whether an object is in the LinkedList
	public boolean contains(Comparable obj) { //O(n)
		ListElement temp = head;
		while (temp != null) {
			if (temp.first() == obj) {
				return true;
			}
			temp = temp.rest();
		}
		return false;
	}
    
	// This method checks whether an object is in the LinkedList and returns it or null if it is not present
	public Object find(Object obj) { //O(n)
		ListElement temp = head;
		while (temp != null) {
			if (temp.first().compareTo(obj) == 0) {
				return temp.first();
			}
			temp = temp.rest();
		}
		return null;
	}   
    
	// This method returns the last element of the LinkedList
	public Object get(int n) { //O(n)
		ListElement d = head;
		while (n > 0) {
			d = d.rest();
			n--;
		}
		return d.first();
	}
	
    // This method prints the LinkedList
	public void print() { //O(n)
		try {
			System.out.println(this.toString());
		} catch (NullPointerException e) {
			System.out.println("The list is empty.");
		}
	}
    
   	// This method returns a String representing the elements in the LinkedList
	public String toString() { //O(n)
		String result = "";
		ListElement temp = head;
		while (temp.rest() != null) {
			result += temp.first().toString() + " ";
			temp = temp.rest();
		}
		result += temp.first().toString();
		return result;
	} 
    
    // This method reverses the LinkedList
	public void reverse() {	//O(n)
		ListElement prev = head;
		ListElement current = head.rest();
		ListElement next = head.rest();
		while (next != null) {
			current = current.rest();
			next.setRest(prev);
			prev = next;
			next = current;
		}
		head.setRest(null);
		head = prev;
	}
    
    // This method interleaves two LinkedLists, it takes an element from the first LinkedLists, then the second, then the first, ...
	public LinkedList interleave(LinkedList secondList) { //O(n)
		LinkedList result = new LinkedList();
		ListElement first = head;
		ListElement second = secondList.head;
		while (first != null & second != null) {
			if (first != null) {
				result.addLast(first.first());
				first = first.rest();
			}
			if (second != null) {
				result.addLast(second.first());
				second = second.rest();
			}
		}
		return result;
	}
    
    // This method creates a sorted LinkedList
	public void addSorted(Comparable o) { //O(n)
		if (head == null)
            // an empty list , add element in front
            head = new ListElement(o, null);
        else if (head.first().compareTo(o) > 0) {
            // we have to add the element in front
            head = new ListElement(o, head);
		} else {
            // we have to find the first element which is bigger
            ListElement temp = head;
			while ((temp.rest() != null) && (temp.rest().first().compareTo(o) < 0)) {
				temp = temp.rest();
			}
			ListElement next = temp.rest();
			temp.setRest(new ListElement(o, next));
		}
		count++;
	}
}

Adding an element to the front of the list

为了在已经存在的列表的开头添加一个元素，需要进行几个操作：

  1. 为新数据制作一个新的列表元素

    making a new list element for the new data

  2. 使列表元素的其余部分指向头部

    making the rest of the list element point to the head

  3. 使头部指向新的列表元素

    make the head point to the new list element

{
    ListElement newElement = new ListElement ( o ); // step 1
    newElement.setRest( head ); // step 2
    head = newElement ; // step 3
}

然而，这三个操作可以在一行 JAVA 代码中执行：

public void addFirst ( Object o )
{
    head = new ListElement(o, head);
}

Accessing an element in the list

为了访问列表中的第 n 个元素，我们只能从头开始并移动 n 次到列表中的下一个元素：

public Object get (int n)
{
    ListElement d = head;
    while (n > 0)
    {
        d = d.rest ();
        n--;
    }
    return d.el1 ;
}

class ListNode{
	int val;	//数值 data
	ListNode next;	// 结点 node
	
	ListNode(int x){	//可以定义一个有参构造方法，也可以定义一个无参构造方法
		val = x;
	}
	// 添加新的结点
	public void add(int newval) {
		ListNode newNode = new ListNode(newval);
		if(this.next == null)
			this.next = newNode;
		else
			this.next.add(newval);
	}
	// 打印链表
	public void print() {
		System.out.print(this.val);
		if(this.next != null)
			{
				System.out.print("-->");
				this.next.print();
			}
	}
}

  8. 链表上的fropple方法对链表进行操作，并每两个连续元素交换一次。 例如，列表 [1 2 3 4 5 6] 被折叠成 [2 1 4 3 6 5]。 这个fropple操作可以在 O(N) 中执行，只需在链表上进行一次迭代。

    详细绘制fropple运算符如何更改链表中的指针。 由于更改指针的顺序至关重要，请对它们进行编号。 在链表类上实现fropple方法。 您的代码不应创建新列表； 应该修改当前列表！ 只遍历列表一次！

    public void fropple() {
    ListElement d = head;
		 
    while(d!=null && d.rest()!=null) {
        Object x = d.el1;
        d.el1 = d.rest().el1;
        d.rest().el1 = x;
        d = d.rest().rest();
    }
}

// This method fropples a LinkedList. This means swapping the first and second element, the third and fourth, etc.
public void fropple2() {	//O(n)
    ListElement odd = head;
    ListElement even = head.rest();
    ListElement storage = head.rest();
    even = even.rest();
    head = head.rest(); 			
    // Start with the second element
    try {
        while (even != null) {
            storage.el2 = odd;
            // Link even index to the previous index
            storage = even;
            even = even.rest();
            odd.el2 = even; 		
            // Link odd index to index + 3
            odd = storage;
            storage = even;
            even = even.rest();
            storage.el2 = odd; 		
            // Link even index to the previous index (necessary to include the very last element)
			}
        odd.el2 = null; // Unlink the now last element from the original last element
    } 
    catch (NullPointerException e) {
        return;
    }
}

public void fropple2() {
    if(count < 2) {
        return;
    }
    ListElement small = head;
    ListElement big = small.rest();
    ListElement next = small.rest();
    head = head.rest();
    while(next != null) {
        next = next.rest();
        big.setRest(small);
        if(next== null ||  next.rest() == null) {
            small.setRest(next);
            return;
        }
        big = next;
        next = next.rest();
        small.setRest(next);
        small = big;
        big = next;
    }
}
