Tuples

A tuple is an ordered sequence of immutable objects. Tuples use parentheses.

Example:

tup1 = ('physics', 'chemistry', 1997, 2000)
tup2 = (1, 2, 3, 4, 5 )
tup3 = "a", "b", "c", "d";
print(tup1)
print(tup2)
print(tup3)

Output:

('physics', 'chemistry', 1997, 2000)
(1, 2, 3, 4, 5)
('a', 'b', 'c', 'd')

Creating a tuple with only one element

x = (9)
print(type(x))
x = (9,)
print(type(x))

Output:

<class 'int'>
<class 'tuple'>

Example:

tup1 = (12, 34.56);

Output:

Tuples are immutable!

Sets

A set is an unordered collection of distinct items.

  • unordered: there is no particular order
  • distinct: any item in the set appears once

Example:

x = {'a','s','d','s'}
print(x)

Output:

{'a', 'd', 's'}

Sets are mutable (i.e. can be changed). They are used when a collection of unique objects is wanted.

Example:

x = set([3, 1, 2, 1])
print(x)

Output:

{1, 2, 3}

Sets are mutable (i.e. can be changed).

Basic operations

In mathematics, set operations include union, intersection, add and remove. All of these (and others) exist in Python.

Example:

ten = set(range(10))
lows = {0, 1, 2, 3, 4}
odds = {1, 3, 5, 7, 9}
lows.add(9)
print(lows)

print(lows.difference(odds))
print(lows.intersection(odds))
print(lows.issubset(ten))
print(lows.issuperset(odds))

lows.remove(0)
print(lows)
lows.union(odds)
print(lows)

{0, 1, 2, 3, 4, 9}

Dictionaries

A dictionary is a collection of values. But unlike indexes for lists, indexes for dictionaries can be of different data types, not just integers. Indexes for dictionaries are called keys, and a key with its value is called a key-value pair.

A dictionary is used to map or associate things you want to store to keys.

Example:

# create a mapping of state to abbreviation
states = {
    'Oregon': 'OR',
    'Florida': 'FL',
    'California': 'CA',
    'New York': 'NY',
    'Michigan': 'MI'
}

conceptually

Example:

# create a mapping of state to abbreviation
states = {'Oregon':'OR', 'Florida':'FL','California':'CA', 'New York':'NY', 'Michigan':'MI'}

print("Michigan's abbreviation is: ", states['Michigan'])
print("Florida's abbreviation is: ", states['Florida'])

Output:

Michigan's abbreviation is: MI
Florida's abbreviation is: FL

Some built-in methods

Method Description
dict.clear() Removes all elements of dictionary dict
dict.get(key) For key key, returns corresponding value
dict.get(key, v) For key key, returns value or default v if the key is not in dict.
dict.keys() Returns a list of dictionary dict's keys.
dict.items() Returns a list of dict's (key, value) tuple pairs.
dict.pop(key) Removes key from dictionary dict and returns value associated to key.
dict.update(d2) Updates dictionary dict with content of dictionary d2
