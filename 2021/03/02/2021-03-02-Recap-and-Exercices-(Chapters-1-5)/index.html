Algebraic proof Exercises

1. Give an equivalence proof for the following theorem:

((p → q) ∧ ¬q) → ¬p

Solution:

((p → q) ∧ ¬q) → ¬p ≡ ((¬p ∨ q) ∧ ¬q) → ¬p Conditional law

​ ≡ ¬[(¬p ∨ q) ∧ ¬q)] ∨ ¬p Conditional law

​ ≡ ¬[[(¬p ∨ q) ∧ ¬q)] ∧ p] De Morgan

​ ≡ ¬[(¬p ∨ q) ∧ (¬q ∧ p)] Associativity

​ ≡ ¬[(¬p ∨ q) ∧ ¬(q ∨ ¬p)] De Morgan

​ ≡ ¬FALSE Contradiction

​ ≡ TRUE

2. Prove each of these equivalences by substitution:

  1. ((p ∨ q) ∧ (¬p ∧ ¬q)) ≡ FALSE
  2. p ∨ q ∨ r ∨ s ≡ (¬p ∧ ¬q ∧ ¬r) → s

Solution 1:

((p ∨ q) ∧ (¬p ∧ ¬q)) ≡ (p ∨ q) ∧ ¬(p ∨ q) De Morgan

​ ≡ FALSE Contradiction

Solution 2:

p ∨ q ∨ r ∨ s ≡ ¬(¬p ∧ ¬q ∧ ¬r) ∨ s De Morgan on three items

​ ≡ (¬p ∧ ¬q ∧ ¬r) → s Conditional law

We also need to prove De Morgan on three items

¬(p ∧ q ∧ r) ≡ ¬((p ∧ q) ∧ r) Associativity

​ ≡ ¬(p ∧ q) ∨ ¬r De Morgan

​ ≡ ¬p ∨ ¬q ∨ ¬r De Morgan

Inference Proof

Previous exam question

Consider the set of statements:

​ “If the band could not play rock music or the refreshments were not delivered on time, then the New Year’s part would have been cancelled and Alicia would have been angry. If the party was cancelled, then refunds would have had to be made. No refunds were made.”

We use these propositional symbols to represent the statements:

  • p: the band could play rock music
  • q: the refreshments were delivered on time
  • r: the New Year/s party was cancelled
  • s: Alicia was angry
  • t: refunds has to be made

You should:

(a) Convert the test statements into propositional statements.

(b) Prove using propositional inference that the band could play rock music,.

Note: You can use the “De Morgen” - laws as inference rules.

Solution

  • i. (¬p ∨ ¬q) → (r ∧ s)
  • ii. r → t
  • iii. ¬t
  1. ¬r Modus tollens with given ii and iii
  2. ¬r ∨ ¬s ∨ introduction with 1
  3. ¬(r ∧ s) De Morgan with 2
  4. ¬(¬p ∨ ¬q) Contrapositive with given i and 3
  5. ¬¬p ∧ ¬¬q De Morgan with 4
  6. p ∧ q Double negation with 5
  7. p ∧ elimination with 6

Strings: exercise 1

Let A and B each be sequences of letters, with A = (a1, a2 ,… , an) and B = (b1,b2 ,… , bn). Describe each of the following situations formally, using quantifiers with respect to In = {1, 2,…, n}, set notation, and the predicates “=”, “≠”, and “≺” (precedes alphabetically) as needed.

  1. There is no occurrence of “c” in A.
  2. Some letter appears twice in A.
  3. No letter appears twice in A.
  4. A and B are the same list.
  5. The set of letters appearing in A is a subset of the set of letters appearing in B.
  6. The set of letters appearing in A is the same as the set of letters appearing in B.
  7. Each element of A is no later in the alphabet than the corresponding element of B.

Solution

  1. There is no occurrence of “c” in A
    • ∀i ∈ In : ai ≠ “c”
  2. Some letter appears twice in A
    • ∃i ∈ In : ∃j ≠ i ∈ In : ai = aj
  3. No letter appears twice in A
    • ∀i ∈ In : ∀j ≠ i ∈In : ai ≠ aj
  4. A and B are the same list
    • ∀i ∈ In : ai = bi
  5. The set of letters in A is a subset of the letters in B
    • ∀i ∈ In : ∃j ∈ In : ai = bj
  6. The set of letters in A is the same as in B
    • Idea: A = B ↔ (A ⊆ B) ∧ (B ⊆ A)
    • Idea: (∀i ∈ In : ∃j ∈ In : ai = bj) ∧ (∀i ∈ In : ∃j ∈ In : aj = bi)
  7. Each element of A is no later in the alphabet than the corresponding element of B
    • ∀i ∈ In : ai ⪯ bi

Strings: exercise 2

Using the notation of Strings Exercise 1, give a formal specification of conditions that are both necessary and sufficient to make A occur earlier in an ordinary dictionary than B. In such a case, we say that A lexicographically precedes B. You may assume that both sequences have the same length, n, and are padded on the right by blanks, which alphabetically precede all the letters. Also note that I1 = {1} and I0 = Ø.

Solution

A lexicographically precedes B if there exists exactly 1 index for which:

  • before this index A and B are equal
  • at this index ai ≺ bi

∃!i ∈ In : (∀j < i : aj = bj ) ∧ ai ≺ bi

Graphs: Exercise 1

  • (a) Express the fact that Edge is symmetric
    • Edge(x, y) ↔ Edge(y, x)
  • (b) Write a formula stating that for every pair of vertices, there is a path of length 2 connecting these two vertices
    • ∀x ∈ V : ∀y ∈ V : ∃z ∈ V : Edge(x, z) ∧ Edge(z, y)
  • (c) Give a recursive definition of Path
    • ∀x ∈ V : ∀y ∈ V : Path(x, y) ↔ x = y ∨ ∃z ∈ V : Edge(x, z) ∧ Path(z, y)

Graphs: Exercise 2

  • (i) ∀x ∈ V : ∀y 6= x ∈ V : Edge(x, y)
    • The graph is complete
  • (ii) ∀x ∈ V : ∀y ∈ V : Path(x, y)
    • The graph is connected
  • (iii) ∀x ∈ V : ∃y ∈ V : Edge(x, y)
    • There are no unconnected vertices
  • (iv) ∃x ∈ V : ∀y 6= x ∈ V : Edge(x, y)
    • There exists a vertex that is directly connected to all other vertices

Edge(a, b), Edge(b, c), Edge(b, d)

  • (i): FALSE

  • (ii): TRUE

  • (iii): TRUE

  • (iv): TRUE

  • (i): FALSE
  • (ii): TRUE
  • (iii): TRUE

Applying logic to mathematics

Exercise

Prove that for any n its successor times its predecessor is 1 less than its square. In other words, prove that ∀n ∈ N : P(n), where P(n) is (n + 1)(n - 1) = n2 - 1. You do not need mathematical induction for this. Use V introduction and arithmetic manipulations involving distributivity and associativity. You may regard other rules of algebra as common knowledge that do not need to be explicitly provided as justifications. Lay out your proof like the rules-of-inference proofs in Chapter 3, justifying each step by stating its rule of inference.

Solution

Prove: ∀n ∈ N : (n + 1)(n − 1) = n2 − 1

  1. [n ∈ N] Assumption
  2. (n + 1)(n − 2) = n2 − n + n − 1 Distributivity
  3. (n + 1)(n − 2) = n2 − 1
  4. ∀n ∈ N : (n + 1)(n − 2) = n2 − 1 ∀ introduction
