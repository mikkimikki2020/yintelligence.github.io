Proving with Predicates

Inference rules

→ Introduction

  • Cf. propositional rule
  • p(x) and q(x) are propositions

If the only assumption about x is that it is a member of U, and if this assumption leads to the truth of q(x), then q must hold for every member of U

If at least one element of a set satisfies a predicate, this justifies the introduction of ∃

∧ Elimination

  • Also called instantiation
  • Generalises propositional ∧-elimination
  • If p is true for all possible arguments, it must be true for a particular one

  • a must not be the name of a particular member of U
  • a should not have been mentioned earlier in the proof

Proof strategies

From: ∃X ∈ U : P(X)

  • Proof by example
    • Show that p(x) is TRUE for a specific x
  • Algorithmic Proof
    • Give an algorithm for finding a specific x for which p(x) is TRUE
    • Prove that the algorithm cannot fail
  • ∀ Introduction
    • Assume x ∈ U and show that p(x) is a consequence
    • Important to assume x is an arbitrary element of U

FORM: ¬∀X ∈ U : P(X)

  • Proof by counterexample
    • Use a De Morgan law
    • (¬∀x ∈ U : p(x)) ≡ (∃x ∈ U : ¬p(x))
    • Proof by example that there exists an x for which ¬p is TRUE

FORM: ∀X ∈ N : P(X)

  • Proof by mathematical induction

Applying logic to mathematics

Introduction

New application domain:

  • New predicates
  • New inference rules

Rules

  • Subset Rule
    • A ⊆ B follows from ∀x ∈ A : (x ∈ B)
  • Set equality rule
    • A = B follows from A ⊆ B and B ⊆ A
  • Transitivity of <
    • a < c follows from a < b and b < c
    • E.g. =, ⊆ and ≡ are transitive as well
  • Pigeon hole principle
    • If n + 1 occurrences are assigned to n categories, there must be one category with at least two occurrences

Example proof

  • Definition:
    • ∀x ∈ N : Odd(x) ↔︎ ∃w ∈ N : (x = 2w + 1)
  • Prove:
    • “If a and b are odd, then a times b is odd”
    • ∀a ∈ N : ∀b ∈ *N : (Odd(a) ∧ Odd(b) → Odd(ab))

Exercise

Prove that for any n its successor times its predecessor is 1 less than its square. In other words, prove that ∀n ∈ N : P(n), where P(n) is (n + 1)(n - 1) = n2 - 1. You do not need mathematical induction for this. Use V introduction and arithmetic manipulations involving distributivity and associativity. You may regard other rules of algebra as common knowledge that do not need to be explicitly provided as justifications. Lay out your proof like the rules-of-inference proofs in Chapter 3, justifying each step by stating its rule of inference.

Solution

Prove: ∀n ∈ N : (n + 1)(n − 1) = n2 − 1

  1. [n ∈ N] Assumption
  2. (n + 1)(n − 2) = n2 − n + n − 1 Distributivity
  3. (n + 1)(n − 2) = n2 − 1
  4. ∀n ∈ N : (n + 1)(n − 2) = n2 − 1 ∀ introduction

Mathematical induction

Introduction

  • Mathematical induction (MI) is used for proving things about infinite sets of propositions
  • Key idea: when trying to prove P(n), we assume that P(n − 1) has already been established
  • This is called the inductive hypothesis

Base case and inductive step

  • A proof by MI has two parts:
    • Base case (BC): typically P(0)
    • Inductive step (IS): typically P(n − 1) → P(n)
  • When we make no assumptions about n except that n ≥ 1 , this is true for any n ≥ 1
  • Repeated use of the inductive step with modus ponens gives us: P(0) (from BC) → P(1) → P(2), → ... → P(n)

Limits of logic

Soundness of rules

  • A rule is sound if it is true in all states, i.e. if it is a tautology
  • Soundness can for example be proved using truth tables
  • Propositional logic:
    • Start set: soundness proven by truth tables
      • E.g. modus ponens, modus tollens, ...
    • Rules proven by inference from this start set
      • E.g. transitivity of the conditional
      • These rules are equally sound

System of logic

A system of logic consists of:

  • A set of axioms
  • A set of inference rules

A system of logic is complete if:

  • All true expressions can be proven

A system of logic is consistent if:

  • It is impossible to prove any expression to be both true and false

Soundness, completeness and consistency

We seek a system:

  • with sound rules
  • that is complete
  • that is consistent

Such systems exist for propositional logic

Such systems do not exist for predicate logic (cf. Gödel’s incompleteness theorems)
