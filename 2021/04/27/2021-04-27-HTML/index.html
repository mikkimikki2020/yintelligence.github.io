HTML Basics

What is HTML?

  • HTML stands for Hyper Text Markup Language 超文本标记语言
  • HTML describes the structure of a Web page 描述了网页的结构
  • HTML consists of a series of elements 由一系列元素组成
  • HTML elements tell the browser how to display the content 元素告诉浏览器如何显示内容
  • HTML elements are represented by tags 元素由标签表示
  • HTML tags are not displayed but used to display their content 标签不显示，但用于显示其内容

What does it look like?

<!DOCTYPE html>
<html>
    <head>
        <title>My title</title>
    </head>
    <body>
        <h1>A heading</h1>
        <a href="#">Link text</a>
    </body>
</html>

HTML Tags 标签

<tag> </tag> OR use closing tags <tag\>

Some tags can only be closing tags (e.g. new line)!

有些标签只能是结束标签（例如换行）！

HTML attributes 属性

<html lang="en-US">

Each tag has their own supported set of attributes

每个标签都有自己支持的属性集

Software

  • Notepad (no colors)
  • Visual Studio Code
  • Your browser (double click the file you made)
  • (Optional) Web Server
    • Not required for basic HTML, CSS, JS development
    • Required when working with external JS files

Exercise

The goal of this exercise is to create and complete an HTML page that looks like the provided pictures in this document. You only have to use pure HTML, so do not use CSS!

Fora complete list and description of the available HTML tags, see w3schools. You will need tags such as <br>, <a>, <h1>, <h2>, <input>, <textarea>, <table>, <ul>, <ol>, <p>, <title>, <img>, <iframe>, ... .

A full list of HTML and HTML5 tags can be found here.

Requirements

  • Editor with HTML syntax highlighting. Visual Studio Code is recommended. Other alternatives include GitHub Atom, Sublime Text, Notepad++, ...

Exercise

Step 1: Creating the HTML file 创建 HTML 文件

For this first exercise or step you are going to create a new blank HTML file calledex3.html.

<!DOCTYPE  html>
<html>
    <head>
    </head>
    <body>
        Hello  World!
    </body>
</html>

Open the file in your browser (double click it), you should see ”Hello World!”. We will continue editing and expanding this web page in the next exercises.

Step 2: Adding titles 添加标题

Using the ex3.html file from the previous exercise, remove the ”Hello World!” text and edit the HTML to visually represent the following result.

Also notice the addition of a title in the browser tab that says ”EX3 - HTML”.

Step 3: Paragraphs

In the following exercises we will expand each section created in the previous exercise with content.

For the first section, create two paragraphs of text. Web developers often use Lorem Ipsum as a placeholder for big portions of text.

Step 4: Hyperlinks

In the next section (Section 2)you will create a link to a website of choice. Make sure to make the text that is displayed to be different than the link.

Step 5: Images

Section 3 will contain an image of your choice. For this exercise you can specify the image size without using CSS.

Step 6: Lists

Create two lists in section 4. The first list is a list with bullets, the second list uses numbers for each item.

Step 7: Tables

For section 5 you have to create a simple table. Use an HTML-encoded symbol for the ’Euro’ sign.

Step 8: Input controls

Create a basic form consisting of at least a text field, text area, radio buttons and a normal button.

Make sure that only one of the radio buttons can be ’checked’. Also make sure that one of the radio buttons is selected by default when you load the page.

Step 9: Website window

Create a window in section 7 that shows another site or PDF document.

Note: This type of HTML component is used less than in the days of Web 1.0. However, it is still often used to display PDF files on a site.

Step 10: HTML Validation

Modern browsers are very forgiving in HTML syntax. They will try to fix or ignore errors such as incorrectly closed tags, quotes or other problems you may overlook. Use the W3 HTML validator to fix all errors and warnings.

Note: You may get asked to use CSS for the table in Step 7, you may ignore this error.

Step 11: Hyperlink with new tab

Change section 2 to open the hyperlink in another windows or tab.

Step 12: Link references

Change the section titles so you can automatically scroll to their location when you add ”#section6”behind the URL.(ex3.html #section6).

Step 13: More input controls

Explore all different input controls in section 6 (e.g. check buttons, hidden, ...).

Step 14: Local images

If you haven’t already, save some images from Google on your PC and use them on your web page. Make sure to put the image files in the same (relative) location as you would on a real website.

Step 15: HTML5 - Video

Create an additional section with a component that can play a video. Note that you should not use a YouTube video but a normal mp4 file that can be played in the browser. Explore the different types of parameters that are available for controlling the video.

Step 16: CSS basics

You will notice the comment in step 10 that says that you should use CSS for the table in step 7.Fix this error by using some CSS.

Hint: Add some simple <style> tags

Step 17: Publish your website

Finally we will publish our website on Wilma (Servers for the WE faculty). Follow the steps in the extra document on Canvas.
