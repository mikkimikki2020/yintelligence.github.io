Circular Vectors

VUB 4013759ENR Algorithms and Data Structures

Slide 108 ~ 116

Circular Vector是对常规 Vector 的一种调整，它避免了addFirst和RemoveFirst方法中所有元素的移位，使得Circular Vector中的两个操作的时间复杂度都是 O(1) 而不是 O(n) 就像常规的向量。 为了获得这种性能增益，需要更复杂的实现。

Circular Vector中的想法是，Circular Vector的第一个元素（即元素getFirst将返回）并不总是存储在底层Vector中的位置 0。相反，索引“start”被存储并表示第一个元素的位置。

上给出了一个例子：最上面的一行显示了一个包含元素 1、2、3、4 的Circular Vector。在这种情况下，“start”变量的值为 0，因为 Vector 从位置 0 开始填充。

第二行显示了removeFirst()操作的结果：Vector 的内容没有向左移动，而是将“start”的值增加到 1。因此，此时对getFirst()的调用将返回 2 作为圆形向量“开始”处的值现在是 2。

同图的第三行显示了通过调用addLast(5)在末尾添加元素 5 后的内部状态。

同图中的最后一行显示了通过调用addLast(6)在末尾添加元素 6 后的内部状态，虽然在最后加上 5 后看起来 Vector 已经满了，但事实并非如此：vector 中所有的开头都有一个空的地方，在位置“start”之前。元素被添加到那里。所以，当到达末尾时，它从头开始添加，它是循环circular的。

  • addFirst
    • Add the new element at position ”start - 1”
    • Unless start is 1, then we need to add at the end of the vector, i.e. at”capacity - 1”.
    • start = (start + capacity - 1) % capacity
  • addLast
    • Add the new element at position ”start + count”
    • When start + count ≥ capacity, then we need to add at the beginning of the vector.
    • Store the element at (start + count) % capacity

Logical versus physical layout

Adding an element to the front

如上所述，Circular Vector上的addFirst与 Vector 上的addFirst不同，因为不会移动任何元素。

如图所示，新元素存储在向量内容的实际开始之前，即位置“start - 1”。 然而，“start - 1”可以变得小于零，如图的第三行。 在这种情况下，元素应该添加到最后一个位置，即“capacity - 1”。 因此，为了用一行代码覆盖这两种情况，我们设置start= (start+capacity−1)%capacity。然后将新元素存储在更新的开始处，

当然，我们需要增加计数变量count variable。 这个公式对于空循环向量的情况是不正确的，因此需要在单独的情况下解决。

Adding an element to the end

addLast类似于addFirst方法。 在这种情况下，元素存储在末尾，即位置“start+count”。 为了避免在vector结束后写入，我们将元素存储在(start+count)%capacity。

Removing an element at the beginning

如果我们简单地将开始加一（模数偏离路线modulo count off course）并将count减一，我们已经将起始位置移动了一个元素，并且Circular Vector的先前第一个元素将被覆盖。

Removing an element at the end

这第四个操作类似于所有其他操作，但更简单。 我们唯一需要做的就是将计数减一。

    public class CircularVector 
{
	private Object data[];
	private int first;
	private int count;
	private int capacity;
	
	public CircularVector(int capacity)
	{
		count = 0;
		first = 0;
		data = new Object[capacity];
		this.capacity = capacity;
	}
	
	public int size()
	{
		return count;
	}
	
	public void print()
	{
		System.out.print(" [");
		for (int i = 0; i < count; i++)
		{
			int index = (first + i + 1) % capacity;
			System.out.print(data[index] + " ");
		}
		System.out.print("] ");
	}
	
	public void addFirst(Object element)
	{
		data[first] = element;
		System.out.println(first);
		if (first == 0) 
		{
			first = (first + capacity - 1) % capacity;
		}
		else 
		{
			first = first - 1;
		}
		count++;
	}
	
	public void addLast(Object element)
	{
		if (first == capacity - 1) 
		{
			first = (first + count - 1) % capacity;
		}
		else 
		{
			first = first + 1;
		}
		data[first - count + 1] = element;
		System.out.println(first);
		count++;
	}
}

    public void reverse(){
    if(head == null) 
        return;
    else 
        if(head.rest() == null) 
            return;
    else{
        ListElement prev = null;
        ListElement current = head;
        ListElement next = head.rest();
        while(prev != tail){
            current.setRest(prev);
            prev = current;
            current = next;
            next = current.rest();
        }
        tail = head;
        head = prev;
        tail.setRest(head);
    }
}
