Function

  1. Write a function that has x and y as parameters. The function should return the solution of the following equation: \[ \left\{ \begin{aligned} (x*y)-x & & if \ x≥y\\ (x*y)-y && if \ x≤y \end{aligned} \right. \]

  2. Write a function that has the variable a as a parameter. This function will call the function from question 1 for all combinations of （i,a） where the parameter i ranges between 𝟏≤𝒊≤𝟐𝒂. The result should be printed every time.

  3. Execute the function from question 2 for the value a = 5.

Matrices

Circuit simulation

We will use Python to simulate the charging of a capacitor. Firstly, you will write the proper function to do this, then,you will use this function to calculate a concrete example.

You are given the following circuit with source V0, resistance R and capacitor C. We want to simulate the voltage over the capacitor （Vc） and the current in the circuit （I）.

The necessary equations to do so are: \[ V_{c} = V_{0} * (1-e^{\frac{-t}{RC}})\\ I=\frac{V_0}{R}*e^{\frac{-t}{RC}} \]

  1. Write a function that calculates the evolution of the voltage and current in this circuit. The function has 5 parameters: V0, R, C, a stopping criterion and a time step for the simulation. The simulation will stop when Vc becomes bigger than a certain fraction of V0 （this fraction, provided as a parameter to the function,will be the stopping criterion）. For each time step the simulation stores t, I and Vc in a nested list. This nested list is returned by your function.

    Tip: to compute an exponential power you can use the function exp from the math library. Import that library on the top of your code and use math.exp(a) to compute 𝑒α.

  2. Use the function from question 1 to simulate Vc and I with the following parameters: V0=12V, R=1kΩ and C=100μF. The simulation stops when the capacitor is 90% charged. The time step should be 1ms or smaller.

  3. To be able to plot the simulation results from question 2, save out your nested list in a csv file. In this file, each line contains the information of the simulation at moment in time, separated by a semicolon “;”.

Hint: you can quickly check your csv file by importing it in Excel and make a scatterplot （see below） or plotting it in Python.

Sieve of Eratosthenes

We will program the Sieve of Eratosthenes, a method for finding all prime numbers until a certain value. The algorithm works as follows:

  1. Create a list of numbers,starting with 2 and until a chosen maximum
  2. Choose the smallest, not yet eliminated,number from the list
  3. Eliminate all multiples of the chosen number （besides the number itself）. Start with eliminating the square of the chosen number.
  4. Repeat steps 2 and 3 until the square of the chosen number is larger than the maximum. The numbers that remain are the prime numbers smaller than the maximum.

You will write a function find_primes(a) that returns out all prime numbers between 2 and a given number a. This function has the following functionality:

  1. Make two lists: the first list contains all number from 2 until a. The other list has the same length as the first list and contains only ones. This second list is used to keep track of the corresponding numbers in the first list, marked as prime numbers. A one in the second list means that the corresponding element in the first list is a prime number, a zero means it is not.

  2. Write the necessary steps 2 and 3 from the algorithm described above. Let the program stop automatically when the square of the chosen number becomes larger than a.

    Note: eliminating a number is performed by changing the corresponding element in the second list to 0.

  3. Print out all non-crossed-out numbers （i.e. all numbers that have a corresponding 1 in the second list）.

    Test your function by printing all prime numbers smaller or equal to 41!

Loan repayment

  1. We will write a function that simulates paying off a loan. Assume we want to take a loan of TOT euro. The interest rate is r% and we can only pay back MAX euro per year. Every year we will have to pay interest （IN）, i.e. r% of the amount left to pay.

    For the first year,the interest IN will be r% of TOT. Given that we can only payback MAX euro per year, we can still pay a capital of （MAX-IN）. After the first year,we will still owe the bank an amount of TOT-（MAX-IN） euros. On this remaining amount,we will have to pay interest again the year after. Since the amount we have to pay back diminishes every year, the interest will become smaller as well.

    For example: you want to take a loan of an amount of 100000 euros. You can pay off a maximum amount of 7000 per year. The current interest rate is 3%.

    The first year we pay 100000 × 0.03 = 3000 euro of interest. Hence we can still pay 7000 - 3000 = 4000 euro of capital and in total we will still need to pay off 100000 - 4000 = 96000 euro.

    *The second year we pay 96000 × 0.03 = 2880 euro of interest, 7000 - 2880 = 4120 euro of capital and we will be left with 96000 - 4210 = 91880 euro to pay off.

  2. Define the necessary variables in which you can store the amount of the loan, the interest rate and the total amount of money you can pay back per year.

  3. Create a function that calculates an interest and a maximum yearly pay-off, based on the loan. Basically ,your function will calculate a pay-off plan. This pay-off plan is a nested list, where each sub-list contains information of one year:

    [interest,capital,remaining]

    Each sub-list contains the interest and the capital that were paid that year, as well as the remaining amount left to pay after that year.

  4. Make a function to which you can provide a pay-off plan and that prints out the information as follows, e.g.:

    Amount loan: 100000
Current interest: 0.03
Year1 | 4000.00 | 3000.00 | 96000.00
Year2 | 4120.00 | 2880.00 | 91880.00...

    The lines above can be printed with a specific formatting expression:

    print('Year{0:2d} | {1:7.2f} | {2:7.2f} | {3:8.2f}'.format( year, capital,interest,remainder))

    Beware! The underlined part can simply be copy-pasted. The parameters of the format function will change depending on the year.

  5. Call the function from question 2 for with parameters TOT=100000, r=3%, MAX=7000 and store the pay-off plan in a new variable. Print the variable by using the function of question 3.

  6. What happens if you only had 3000 euro per year to pay off your loan? Print a message in case the situation occurs and return an empty pay off list.
