Proving Things: Why and How

VUB 4005196EER Fundamentals of Computer Science

Why?

  • Reasoning allows us to expand our knowledge and beliefs
  • In order to yield correct results, reasoning should be sound
  • Proofs are a form of particularly careful reasoning in the form of a sequence of justified steps

What?

  • A proof is a sequence of assertions meant to convince the reader of the truth of a statement
  • Proofs convince by:
    • starting from agreed truths: axioms
    • continuing by agreed methods: inference rules
  • A proved statement is called a theorem
  • Similar to ordinary reasoning, but symbols are used for precision

Propositional logic 命题逻辑

Propositional logic (PL) is used for all aspects of the proof:

  • Axioms and theorems are propositions

    公理和定理是命题

  • Rules of inference are based on PL

    推理规则基于 PL

Three kinds of proofs

Proof by Truth Table 真值表证明

  • justified by definitions of operators

    由运算符的定义证明

  • justified by exhaustive case analysis

    通过详尽的案例分析证明是合理的

Proof by Substitution 替换证明

  • justified by equivalence

    等价证明

  • also called ‘algebraic proof’

    也称为“代数证明”

Proof by Inference Rules 推理规则证明

  • justified by rules proved within logic

    由逻辑中证明的规则证明

  • often mixed with substitution

    经常与替代混合使用

Rules of inference

General idea

Need to proof premises → conclusion

α1 ∧ α2 ∧ ... ∧ αn → β

Recall implication truth table:

  • When α → β and α = TRUE, β cannot be FALSE
  • If α = TRUE when β = FALSE, we do not have α → β

Importantly, each rule of inference is equivalent to a PL expression and can be evaluated using truth tables.

重要的是，每条推理规则都相当于一个 PL 表达式，可以使用真值表进行评估。

Without assumptions

Checking a proof

  • Proof = list of assertions

    证明 = 断言列表

  • Each assertion needs to be checked

    每个断言都需要检查

Checking an assertion

  • Find an inference rule that matches the conclusion being considered

    找到与所考虑的结论相匹配的推理规则

  • Find earlier assertions that match the premises of the inference rule

    查找与推理规则的前提匹配的较早断言

Example 1

Prove that given (p ∨ q) ∧ ¬p is TRUE that q is TRUE as well

  1. (p ∨ q) ∧ ¬p given
  2. p ∨ q ∧ elimination, with line 1
  3. ¬p ∧ elimination, with line
  4. p → q vacuous proof, with line 3
  5. q → q tautology
  6. q case analysis, with lines 2, 4, 5
Alternative elimination

We proved the validity of the following inference rule:

(p ∨ q) ∧ ¬p → q

Once a theorem has been proved, it can be used in other proofs

Example 2

Prove that given p ∧ ¬p is TRUE that q is TRUE as well

  1. p ∧ ¬p given
  2. ¬p ∧ elimination, with line 1
  3. p → q vacuous proof, with line 2
  4. p ∧ elimination, with line 1
  5. q modus ponens, with lines 3, 4

If p ∧ ¬p are taken to be both TRUE, then any proposition can be proven.

This outcome would be bizarre and undesirable

Assumptions

Principle

In proofs, assumptions can be introduced tentatively, to test their consequences

假设可以试探性地推出，以测试他们的后果

When introduced, we do not know whether an assumption is TRUE

引入时，我们不知道假设是否为真

Assumptions are written between [ ]

假设写在 [ ] 之间

Absurdity and implication

  • Reduction to absurdity
    • [p] ∧ FALSE → ¬p
  • Introduction of implication
    • [p] ∧ q → (p → q)

Nested assumptions

Prove: ((p → q) ∧ (q → r)) → (p → r)

Proving the modus tollens using inference

Inference proof

For theorem analogous to conditional law

Prove: (p → q) ↔︎ (¬p ∨ q)

We have proven:

  • (p → q) ← (¬p ∨ q)
  • (p → q) → (¬p ∨ q)

This implies:

  • (p → q) ↔︎ (¬p ∨ q)

Proof strategies

Proving a conditional

p → q

  • Direct proof:
    • Assume: p = TRUE
    • Show: q = TRUE
    • Therefore: p → q (→ introduction)
  • Indirect proof:
    • Assume: q = FALSE
    • Show: p = FALSE
    • Therefore: ¬q → ¬p (→ introduction)
    • p → q (contrapositive law)
  • Proof by contradiction:
    • Assume: p → q = FALSE
    • Show that this assumption leads to a contradiction

Proving a biconditional

p ↔︎ q

  • Two-way proof:
    • Prove: p → q
    • Prove: p ← q
  • Special case of two-way proof:
    • Set equality
    • A theorem that states that set A and set B are equal can be written as: x ∈ A ↔︎ x ∈ B

Proofs with disjunctions

p ∨ q

  • Case analysis I:
    • Given: p ∨ q
    • To prove: r
    • Prove that r holds in either case: prove p → r and q → r
  • Case analysis II:
    • Given: p ∨ q
    • To prove: r ∨ s
    • Prove: p → r and q → s
  • Excluded middle:
    • For either kind of case analysis, it may be helpful to introduce the tautology u ∨ ¬u
    • Prove p → u and q → ¬u
  • Eliminating an alternative:
    • Given: p ∨ q
    • Eliminate: p
    • Therefore: q = TRUE
