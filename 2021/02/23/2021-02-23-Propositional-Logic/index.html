Exercise

Propositions 命题

A proposition = a statement that is either true or false.

For example:

5 > 3 true
FISH ⊂ MAMMALS false
It is raining either true or false

Logical constraints and variables 逻辑约束和变量

  • Set of logical (boolean) constants: {TRUE, FALSE}

A logical (boolean) variable is a variable that can take only the value TRUE or FALSE.

We use Σ for the set of Boolean variables.

E.g.: when p represents the idea that 2 is smaller than 3, we can write: p = TRUE

  • p = TRUE

Logical operators 逻辑运算符

A proposition can be

  • a logical constant 逻辑常数

  • a logical variable 逻辑变量

  • an expression combining logical constants and/or logical variables with logical operators (i.e. and - inclusive or - not)

    将逻辑常量和/或逻辑变量与逻辑运算符组合在一起的表达式（即和 - 包含或 - 不包含）

Operator symbols 运算符符号

In English, the words and, or and not are ambiguous

In order to avoid these ambiguities, the following symbols are used for logical operators

and ∧
inclusive or ∨
not ¬
Example propositions
  • Let p: the proposition that it is raining
  • Let q: the proposition that 2 is smaller than 3

Propositions:

p it is raining
p ∧ q it is raining and 2 is smaller than 3
p ∨ q it is raining or 2 is smaller than 3
¬p it is not raining

Defining propositions 定义命题

Definition: Propositions:

  • Each of the two logical constants, TRUE and FALSE is a proposition.

    两个逻辑常量TRUE和FALSE中的每一个都是一个命题。

  • If p ∈ Σ, then p is a proposition.

    如果 p ∈ Σ，则 p 是一个命题。

  • If α and β are propositions, then so are (α ∧ β), (α ∨ β), and (¬α).

    如果 α 和 β 是命题，那么 (α ∧ β)、(α ∨ β) 和 (¬α) 也是命题。

Note:

  • The definition is recursive 定义是递归的

  • Parentheses are used to avoid ambiguity 括号用于避免歧义

  • This definition defines the syntax of propositional logic

    这个定义定义了命题逻辑的语法

State, operators and truth tables

State 状态

State of a proposition (e.g. p)

  • Its value
  • 2 possible states (TRUE or FALSE)

State of a collection of propositions (e.g. p, q)

  • An ordered list of the states of each proposition in the collection

    集合中每个命题的状态的有序列表

  • 2k possible states (with k = number of propositions in the collection)

More commonly called interpretation, assignment, structure, possible world and defined as a function Σ → {TRUE, FALSE}.

更常见的称为解释、赋值、结构、可能世界并定义为函数Σ → { TRUE , FALSE }。

p q
TRUE TRUE
TRUE FALSE
FALSE TRUE
FALSE FALSE

  • p ∧ q = TRUE when both p = TRUE and q = TRUE
  • p ∧ q = FALSE in the other three states

¬ (not, negation)

  • ¬p = TRUE when p = FALSE
  • ¬p = FALSE when p = TRUE

Evaluating Propositions

Let I : Σ → {TRUE, FALSE} be an interpretation.

  • TRUE and FALSE are propositions. The value of each of them in I is itself.
  • If p ∈ Σ, then the value of p in I is I(p).
  • If α and β are propositions, then
    • (¬α) is FALSE in I if α is TRUE in I and vice-versa.
    • (α ∧ β) is TRUE in I if both α and β are TRUE in I, and otherwise is FALSE in I.
    • (α ∨ β) is FALSE in I if both α and β are FALSE in I, and otherwise is TRUE in I.

Consider the effect of parentheses.

Give a set of truth values for p, q, and r for which the values of (p ∨ q) ∧ r and p ∨ (q ∧ r) are different.

Be lazy, generate models (interpretations that make a proposition true) using the IDP system!

Terminology:

  • Vocabulary: Σ
  • Theory: set of propositions
  • Structure: Interpretation
  • Model expansion: Procedure that searches interpretations that make a theory TRUE.

Solution

  • {p = TRUE, q = FALSE, r = FALSE}
  • {p = TRUE, q = TRUE, r = FALSE}

Proofs with truth tables

Equivalence 等价

Two propositional expressions that have the same value for each and every state are said to be equivalent

对于每个状态都具有相同值的两个命题表达式被称为等价的

Proof by exhaustive case analysis:

Equivalence between α and β is notated as α ≡ β, so ¬(p ∧ q) ≡ ¬p ∨ ¬q

Tautology 同义反复

A tautology is a proposition that is equivalent to TRUE

A tautology is true in every assignment, so its column in the truth table contains only TRUEs

重言式在每个赋值中都是真的，所以它在真值表中的列只包含TRUE

Validity, satisfiability, unsatisfiability

A propositional expression is

  • valid if its truth table column has all TRUEs, in which case it is a tautology

    有效，如果它的道理表列有所有 TRUE小号，在这种情况下，它是一个同义反复

  • satisfiable if its truth table column has some TRUEs (there exists an assignment that satisfies it)

    可满足的，如果它的道理表列有一些 TRUE小号（存在一个分配是满足它）

  • unsatisfiable if its truth table column has all FALSEs, in which case it is a contradiction

    不可满足的，如果它的道理表列有所有 FALSE小号，在这种情况下，它是一个矛盾

Laws of Propositional Logic 命题逻辑定律

Equivalences with one variable

  • Law of negation 否定定律: ¬¬p ≡ p

  • Combining a variable with itself:

p ∨ ¬p ≡ TRUE Excluded middle
p ∧ ¬p ≡ FALSE Contradiction
p ∨ p ≡ p Idempotence of ∨
p ∧ p ≡ p Idempotence of ∧
  • Properties of constants 常量的属性:
p ∨ TRUE ≡ TRUE
p ∨ FALSE ≡ p
p ∧ TRUE ≡ p
p ∧ FALSE ≡ FALSE

Equivalences with multiple variables

  • Commutativity 交换性:
    • p ∧ q ≡ q ∧ p
    • p ∨ q ≡ q ∨ p
  • Associativity 关联性:
    • p ∨ (q ∨ r) ≡ (p ∨ q) ∨ r
    • p ∧ (q ∧ r) ≡ (p ∧ q) ∧ r
  • Distributivity 分配性:
    • p ∨ (q ∧ r) ≡ (p ∨ q) ∧ (p ∨ r)
    • p ∧ (q ∨ r) ≡ (p ∧ q) ∨ (p ∧ r)
  • DeMorgan’s laws 德摩根定律:
    • ¬(p ∧ q) ≡ ¬p ∨ ¬q
    • ¬(p ∨ q) ≡ ¬p ∧ ¬q
  • Subsumption 包含:
    • p ∧ (p ∨ q) ≡ p

Exercise

Prove the following law by DeMorgan using a truth table

  • ¬(p ∨ q) ≡ ¬p ∧ ¬q

Solution

Truth table:

p q p ∨ q ¬(p ∨ q) ¬p ¬q ¬p ∧ ¬q
T T T F F F F
T F T F F T F
F T T F T F F
F F F T T T T
  • ¬(p ∨ q) ≡ ¬p ∧ ¬q

Two important operators

Implication

→ (if then, implies, conditional operator, implication 如果那么，暗示，条件运算符，暗示)

  • p → q = FALSE when p is TRUE and q is FALSE
  • p → q = TRUE in all other states
→ q

Biconditional

↔︎ (if and only if, biconditional operator)

  • p ↔︎ q = TRUE when p and q have the same value
  • p ↔︎ q = FALSE when p and q have a different value
↔︎ q
p T F
T T F
F F T

Called biconditional operator because of the following equivalence:

  • p ↔︎ q ≡ (p → q) ∧ (q → p)

Equivalences with the conditional and biconditional

  • Conditional law: p → q ≡ ¬p ∨ q
  • Biconditional law: p ↔︎ q ≡ (p → q) ∧ (q → p)
  • Contrapositive law: p → q ≡ ¬q → ¬p

  • ((p → q) ∧ (q → r)) → (p → r)

3. Prove by substitution that the following expression is a tautology

  • p ∨ (¬p ∧ q) ≡ (p ∨ q)

Solution

p ∨ (¬p ∧ q) ≡ (p ∨ ¬p) ∧ (p ∨ q) Distributive law

p ∨ (¬p ∧ q) ≡ TRUE ∧ (p ∨ q) Excluded middle

p ∨ (¬p ∧ q) ≡ (p ∨ q) Property of constant

Normal forms 范式

Conjunctive normal form 合取范式 (CNF)

An expression is said to be in conjunctive normal form, if

  • it is a conjunction of disjunctions, and

  • the negation operator only applies to logic variables

    否定运算符仅适用于逻辑变量

E.g. convert p ↔︎ q into CNF:

p ↔︎ q ≡ (p → q) ∧ (q → p) Biconditional law

p ↔︎ q ≡ (¬p ∨ q) ∧ (¬q ∨ p) Contrapositive law

Disjunctive normal form 析取范式 (DNF)

An expression is said to be in disjunctive normal form, if it is a disjunction of conjunctions of logic variables and negated logic variables

≡ (¬p ∨ q) ∧ (¬q ∨ p)

≡ (¬p ∧ ¬q) ∨ (¬p ∧ p) ∨ (q ∧ ¬q) ∨ (q ∧ p) Distributive law

≡ (¬p ∧ ¬q) ∨ FALSE ∨ FALSE ∨ (q ∧ p) Excluded middle

≡ (¬p ∧ ¬q) ∨ (q ∧ p) Properties of constant

Exercise

4. The operator ⊕ expresses the idea of exclusive or, that is, a choice between alternatives. Thus p ⊕ q is TRUE when exactly one of p, q is TRUE, but is FALSE when both or neither is TRUE

  • Give the shortest possible equivalence for p ⊕ q
  • Give an expression equivalent to p ⊕ q containing only ∧, ∨, and ¬ as its operators

Solution

  • ¬(p ↔︎ q)
  • (p ∧ ¬q) ∨ (¬p ∧ q)

5. Prove distributivity from the right of ∧ over ∨, algebraically. That is, use a sequence of equivalences to show that

  • p ∨ q) ∧ r ≡ (p ∧ r) ∨ (q ∧ r)

Solution

(p ∨ q) ∧ r ≡ r ∧ (p ∨ q) Commutativity

(p ∨ q) ∧ r ≡ (r ∧ p) ∨ (r ∧ q) Distributivity

(p ∨ q) ∧ r ≡ (p ∧ r) ∨ (q ∧ r) Commutativity
