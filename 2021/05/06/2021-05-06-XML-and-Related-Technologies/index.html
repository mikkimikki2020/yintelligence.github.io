XML

What is XML?

  • Standardised text format for (semi-)structured information

    （半）结构化信息的标准化文本格式

  • Meta markup language 元标记语言

    • tool for defining other markup languages

      定义其他标记语言的工具

      • e.g. XHTML, WML, VoiceXML, SVG, Office Open XML (OOXML)

  • Data surrounded by text markup that describes the data

    被描述数据的文本标记包围的数据

    • ordered labelled tree
<note date="2020-11-05">
    <to>Maxim Van de Wynckel</to>
    <from>Beat Signer</from>
    <content>Let us discuss exercise 7 this afternoon ...</content>
</note>

What is Not XML?

  • XML is not a programming language

    XML不是编程语言

    • however, it can be used to represent program instructions, configuration files etc.

      但是，它可以用来表示程序指令、配置文件等。

    • note that there is an XML application (XSLT) which is Turing complete

      请注意，有一个图灵完备的 XML 应用程序 (XSLT)

  • XML is not a database

    XML不是数据库

    • XML is often used to store long-term data but it lacks many database management system (DBMS) features

      XML 通常用于存储长期数据，但缺乏许多数据库管理系统 (DBMS) 功能

    • many existing databases offer an XML import/export

      许多现有数据库提供 XML 导入/导出

    • more recently there also exist native XML databases

      最近还存在原生 XML 数据库

      • e.g. BaseX or eXist

Evolution of XML 演变

  • Descendant of Standard Generalized Markup Language (SGML)

    标准通用标记语言 (SGML) 的后代

    • SGML is more powerful but (too) complex

      SGML 更强大但（太）复杂

  • HTML is an SGML application 一个 SGML 应用程序

  • XML was developed as a “SGML-Lite” version

    XML 被开发为“SGML-Lite”版本

    • XML 1.0 published in February 1998

  • Since the initial XML release numerous associated standards have been published

    自从最初的 XML 发布以来，已经发布了许多相关的标准

Why has XML been so Successful?

  • Simple
  • General
  • Accepted
  • Many associated standards
  • Many (freely) available tools

XML Specification 规范

  • Provides a grammar for XML documents in terms of

    为 XML 文档提供语法

    • placement of tags 标签的放置
    • legal element names 合法元素名称
    • how attributes are attached to elements 属性如何附加到元素
    • ...

  • General tools 通用工具

    • parsers that can parse any XML document regardless of particular application tags

      可以解析任何 XML 文档而不考虑特定应用程序标签的解析器

    • editors (e.g. XMLSpy) and various programming APIs

      编辑器（例如 XMLSpy）和各种编程 API

    Specification available at here

XML Tree Document Structure 树文档结构

An XML document tree can contain 7 types of nodes

  1. root node

    • always exactly one root node

  2. element nodes

    • element node with optional attribute nodes

      具有可选属性节点的元素节点

  3. attribute nodes 属性节点

    • name/value pairs

  4. text nodes 文本节点

    • text belonging to an element or attribute

  5. comment nodes (<!-- ... -->) 评论节点

  6. processing instruction nodes 处理指令节点

    • pass information to a specific application via <? ... ?>

  7. namespace nodes 命名空间节点

Well-Formedness and Validity 格式良好和有效性

An XML document is well-formed if it follows the rules of the XML specification

如果 XML 文档遵循 XML 规范的规则，则它 是格式良好的

  • correct nesting, only valid names, attributes in quotes, …

    正确的嵌套，只有有效的名称，引号中的属性，...

An XML document can be valid according to its Document Type Definition (DTD) or XML Schema

XML文档可以是有效根据其文档类型定义（DTD）或XML模式

  • completely self-describing about its structure and content through

    完全自我描述其结构和内容

    • the document content 文件内容

    • auxiliary files referred to in the document

      文档中引用的辅助文件

  • validity can be checked by a validating XML parser

    有效性可以通过验证 XML 解析器来检查

    online validation service available at here

Differences Between XML and HTML

  • XML is a tool for specifying markup languages rather than a markup language itself

    XML 是用于指定标记语言而不是标记语言本身的工具

    • specify “special markup languages for special applications”

      指定“特殊应用的特殊标记语言”

  • XML is not a presentation language

    XML 不是一种表示语言

    • defines content rather than presentation 定义内容而不是呈现

    • HTML mixes content, structure and presentation

      HTML 混合了内容、结构和表现形式

  • XML was designed to support a number of applications and not just web browsing

    XML 旨在支持多种应用程序，而不仅仅是 Web 浏览

  • XML documents should be well-formed and valid

    XML 文档应该格式正确且有效

    • XML documents are easier to process by a program (parser)

      XML 文档更容易被程序（解析器）处理

  • Readability is more important than conciseness

    可读性比简洁更重要

    • e.g. <tablerow> rather than <tr>

  • Matching of tags is case sensitive

    标签匹配区分大小写

    • e.g. start tag <Bold> does not match end tag </BOLD>

  • Markup requires matching start and end tags

    标记需要匹配的开始和结束标签

    • e.g. <p> and </p>

    • exceptions are special non-enclosing tags

      例外是特殊的非封闭标签

      • e.g. <br/> or <image .../>

  • Whitespaces in texts are significant

    文本中的空格很重要

XHTML is a reformulation of HTML to make it an XML application

XHTML 是 HTML 的重新表述，使其成为 XML 应用程序

  • we accept that HTML is here to stay

  • improve HTML by using XML (with minimal effort)

    使用 XML 改进 HTML（以最少的努力）

Differences Between XHTML and HTML

  • Documents must be valid 文件必须有效

  • XHTML namespace must be declared in <html> element

    XHTML 命名空间必须在<html>元素中声明

  • <head> and <body> elements cannot be omitted 不能省略

  • <title> element must be the first element in the <head>

  • End tags are required for non-empty clauses 非空子句需要结束标记

    • empty elements must consist of a start-tag and end-tag pair or an empty element (e.g. <br/>)

      空元素必须由开始标签和结束标签对或空元素组成（例如<br/>）

  • Element and attribute names must be in lowercase

    元素和属性名称必须小写

  • Attribute values must always be quoted

    必须始终引用属性值

  • Attribute names cannot be used without a value

    没有值就不能使用属性名称

XML Technologies

Overview of XML Technologies 概述

  • XPath and XPointer

    • addressing of XML elements and parts of elements

      XML 元素和部分元素的寻址

  • XSL (Extensible Stylesheet Language 可扩展样式表语言)

    • transforming XML documents (XSLT and XSL:FO)

      转换 XML 文档（XSLT 和 XSL:FO）

  • XLink (XML Linking Language 链接语言)

    • linking in XML

  • XQuery (XML Query Language 查询语言)

    • querying XML documents

  • Document Type Definition 文档类型定义 (DTD) and XML Schema

    • definition of schemas for XML documents

      XML 文档的模式定义

      • DTDs have a limited expressive power

        DTD 的表达能力有限

      • XML Schema introduces datatypes, inheritance etc

        XML Schema 引入了数据类型、继承等

  • SAX (Simple API for XML)

    • event-based programming API for reading XML documents

      用于读取 XML 文档的基于事件的编程 API

  • DOM (Document Object Model 文档对象模型)

    • programming API to access and manipulate XML documents as tree structures

      编程 API 以树结构形式访问和操作 XML 文档

  • RDF (Resource Description Framework 资源描述框架)

    • specific XML encoding used by the semantic web

      语义网使用的特定 XML 编码

Document Object Model 文档对象模型 (DOM)

  • Defines a language neutral API for accessing and manipulating XML documents as a tree structure

    定义一种语言中立的 API，用于以树结构形式访问和操作 XML 文档

    • have already seen the HTML DOM model

  • The entire document must be read and parsed before it can be used by a DOM application

    在DOM 应用程序可以使用之前， 必须读取并解析整个文档

    • DOM parser not suited for large documents!

      DOM 解析器不适合大文档！

  • Two different types of DOM Core interfaces for accessing supported content types

    两种不同类型的 DOM Core 接口，用于访问支持的内容类型

    • generic node interface 通用节点接口

    • node type-specific interfaces (for each of the 7 node types)

      特定于节点类型的接口（对于 7 种节点类型中的每一种）

  • Various available DOM parsers 各种可用的 DOM 解析器

    • e.g. JDOM parser specifically for Java

      例如，专用于 Java 的 JDOM 解析器

XPath

  • Expression language to address elements of an XML document (used in XPointer, XSLT and XQuery)

    用于处理 XML 文档元素的表达式语言（在 XPointer、XSLT 和 XQuery 中使用）

  • A location path is a sequence of location steps separated by a slash (/)

    位置路径是由斜线 (/) 分隔的一系列位置步骤

    • various navigation axes such as child, parent, following etc.

      各种导航轴，例如子、父、跟随等

  • XPath expressions look similar to file pathnames

    XPath 表达式看起来类似于文件路径名

/publications/publication
/publications/publication[year>2008]/title
//author[3]
//title[@lang='eng']

XML Pointer Language (XPointer)

  • Address points or ranges in an XML document

    寻址点或范围

  • Uses XPath expressions

  • Introduces addressing relative to elements

    介绍相对于元素的寻址

    • supports links to points without anchors

      支持指向没有锚点的链接

URI#xpointer(publications/publication[1]) // relative to URI
URI#xpointer(publications/publication[1]/range-to(
publications/publication[3]/howpublished) // range

XML Linking Language (XLink)

  • Standard way for creating links in XML documents, fixes limitations of HTML links where

    在 XML 文档中创建链接的标准方法， 修复了 HTML 链接的限制，其中

    • anchors must be placed within documents

      锚点必须放置在文档中

    • only entire documents or predefined marks (#) can be linked

      只能链接整个文档或预定义标记 ( #)

    • only one-to-one unidirectional links are supported

      仅支持一对一的单向链路

  • XLinks can be defined in separate documents 可以在单独的文档中定义

    • third-party link (metadata) server

      第三方链接（元数据）服务器

  • Two types of links

    1. simple links 简单链接

      • associate exactly one local and one remote resource (similar to HTML links)

        准确关联一个本地资源和一个远程资源（类似于 HTML 链接）

    2. extended links 扩展链接

      • associate an arbitrary number of resource

        关联任意数量的资源

  • other attributes

    • xlink:show: new, replace, embed
    • xlink:actuate: onLoad, onRequest

  • Other XLink features

    • linking parts of resources 链接部分资源
    • typed links 输入链接

  • The Annotea project uses XLink for managing external annotations

    • for example used in the Amaya Web Browser

  • New Microsoft Edge browser

    • annotation of arbitrary webpages

Simple API for XML (SAX)

  • Event-based API for XML document parsing

    用于 XML 文档解析的基于事件的 API

    • many free SAX parsers available (e.g. Apache Xerces)

  • Scans the document from start to end 从头到尾扫描文档

    • invokes callback methods 回调方法

  • Different kinds of events

    • start of document
    • end of document
    • start tag of an element
    • end tag of an element
    • character data
    • processing instruction

  • SAX parser needs less memory than DOM parser

    SAX 解析器比 DOM 解析器需要更少的内存

    • DOM parser often uses SAX parser to build the DOM tree

      DOM 解析器经常使用 SAX 解析器来构建 DOM 树

XML Transformations 格式转换

Developers want to be able to transform data from one format to another

开发人员希望能够将数据从一种格式转换为另一种格式

  • processing of XML documents
    • XML to XML transformation
  • post-processing of documents
    • e.g. XML to XHTML, XML to WML, XML to PDF, ...

The Extensible Stylesheet Language Transformations (XSLT) language can be used for that purpose

在可扩展样式表语言转换（XSLT）语言可以用于这一目的

XSLT Processor

The XSLT processor (e.g. Xalan) applies an XSLT stylesheet to an XML document and produces the corresponding output document

XSLT 处理器（例如 Xalan）将 XSLT 样式表应用于 XML 文档并生成相应的输出文档

XSL Transformations (XSLT)

  • Most important part of XSL

    XSL 最重要的部分

    • uses XPath for the navigation

      使用 XPath 进行导航

  • XSLT is an expression-based language based on functional programming concepts

    XSLT 是一种基于函数式编程概念的基于表达式的语言

  • XSLT uses

    • pattern matching to select parts of documents

      模式匹配以选择文档的一部分

    • templates to perform transformations

      执行转换的模板

  • Most web browsers support XSLT

    • transformation can be done on the client side based on an XML document and an associated XSLT document

XML Applications 应用

XML for Data Interchange 数据交换

  • Standard representation to exchange information between different systems

    在不同系统之间交换信息的标准表示

  • General way to query data from different systems

    从不同系统查询数据的一般方法

    • e.g. via the XML Query (XQuery) language

  • Connect applications running on different operating systems and computers with different architectures

    连接运行在不同操作系统和不同架构计算机上的应用程序

    • XML Remote Procedure Call (XML-RPC)
    • Simple Object Access Protocol (SOAP) which is a successor of XML-RPC and used for accessing Big Web Services

XML Remote Procedure Call (XML-RPC) 远程过程调用

XML-RPC specification released in April 1998

Advantages

  • XML-based lingua franca understood by different applications

    不同应用程序理解的基于 XML 的通用语言

  • HTTP as carrier protocol

    HTTP 作为载体协议

  • not tied to a single object model (as for example in CORBA)

    不绑定到单个对象模型（例如在 CORBA 中）

  • easy to implement (based on HTTP and XML standards)

    易于实现（基于 HTTP 和 XML 标准）

  • lightweight protocol 轻量级协议

  • built-in error handling 内置错误处理

Disadvantages

  • slower than specialised protocols that are used in closed networks

    比封闭网络中使用的专用协议慢

  • Trend for ubiquitous information services 泛在信息服务的趋势

      小屏幕、键盘等。通常使用起来很笨拙

  • Sometimes it is necessary to have hand-free interfaces

    有时需要免提接口

      例如在驾驶或操作机器时

VoiceXML Architecture

Basic VoiceXML Concepts

  • Dialogue 对话

    • conversational state in a form or menu

      表单或菜单中的会话状态

      • form 表单

        • interaction that collects values for field item variables

          收集字段项变量值的交互

      • menu 菜单

        • presents user with a choice of options

          为用户提供选项选择

        • transition to next dialogue based on choice

          根据选择过渡到下一次对话

  • Input

    • recognition of spoken input (or recording of spoken input)

      识别语音输入（或记录语音输入）

    • recognition of DTMF (dual-tone multi-frequency) input

      DTMF（双音多频）输入识别

  • Output

    • speech synthesis 语音合成 (TTS)
    • recorded audio files 录制的音频文件

Other XML Applications

  • Synchronized Multimedia Integration Language (SMIL)

    同步多媒体集成语言

    • animations (timing, transitions etc.)

      动画（时间、过渡等）

  • Mathematical Markup Language (MathML)

    数学标记语言

    • mathematical notations (content and structure)

  • Scalable Vector Graphics (SiVG)

    • two-dimensional vector graphics (static or dynamic)

      二维矢量图形（静态或动态）

  • Ink Markup Language (InkML )

    • digital ink representation (e.g. from digital pen)

      数字墨水表示（例如来自数字笔）

  • Office Open XML (OOXML)

    file format (ZIP) for representing word processing documents, presentations etc. (e.g. *.docx, *.pptx and *.xlsx)

    用于表示文字处理文档、演示文稿等的文件格式

    • various XML files within these ZIP documents

      这些 ZIP 文档中的各种 XML 文件

    • specific markup languages for different domains (wordprocessingML, presentationML, spreadsheetML, …)
