Comparison of Web Application Frameworks

A web application framework is a software framework that is designed to support the

Web应用程序框架是一个软件框架，旨在支持

  • development of dynamic websites 动态网站的发展
  • web applications 应用程序
  • web services 服务
  • web resources 网络资源

The framework aims to alleviate the overhead associated with common activities performed in web development.

该框架旨在减轻与 Web 开发中执行的常见活动相关的开销。

For example, many frameworks provide libraries for database access, templating frameworks and session management, and they often promote code reuse.

例如，许多框架提供用于数据库访问、模板框架和会话管理的库，并且它们经常促进代码重用 code reuse

  • A web application framework offers libraries and tools to deal with web application issues

    Web 应用程序框架提供了处理 Web 应用程序问题的库和工具

    • template libraries, security & session management, database access libraries (ORM) etc.

  • Some frameworks also offer an abstraction from the underlying enabling technologies

    一些框架还提供了对底层支持技术的抽象

    • e.g. automatic creation of Java Servlets

      例如自动创建 Java Servlets

  • Many frameworks follow the Model-View-Controller (MVC) design pattern

    许多框架遵循模型-视图-控制器 (MVC)设计模式

    • no mix of application logic and view (e.g. not like in JSP)

      没有混合应用程序逻辑和视图（例如不像在 JSP 中）

    • increases modularity and reusability

      增加模块化和可重用性

  • Lead to a faster and more robust development process

Model-View-Controller (MVC)

MVC模式（Model–view–controller）是软件工程中的一种软件架构模式，把软件系统分为三个基本部分：模型（Model）、视图（View）和控制器（Controller）。

This is done to separate internal representations of information from the ways information is presented to and accepted from the user.

这样做是为了将信息的内部表示与向用户呈现和接受信息的方式分开。

  • Model 模型

    • data (state) and business logic

      数据（状态）和业务逻辑

    • multiple views can be defined for a single model

      可以为单个模型定义多个视图

    • when the state of a model changes, its views are notified

      当模型的状态发生变化时，会通知其视图

  • View 视图

    • renders the data of the model

      渲染模型的数据

    • notifies the controller about changes

      通知控制器有关更改

  • Controller 控制器

    • processes interactions with the view 处理与视图的交互

    • transforms view interactions into operations on the model (state modification)

      将视图交互转换为对模型的操作（状态修改）

Choosing a Client-side Framework 选择合适的客户端框架

  • Support and maintenance 支持和维护

    • who maintains it? 谁维护它？
    • what companies use it? ($$$)
    • how regular is it updated? 它的更新频率如何？

  • Learning curve & adoptability 学习曲线和可采用性

    • is it similar to another framework?

      它类似于另一个框架吗？

    • is there a clear separation of concerns?

      是否有明确的关注点分离？

  • Performance 表现

    • framework size 框架大小

      • bigger framework = more downloading = slower loading

        更大的框架 = 更多的下载 = 更慢的加载

    • code size 代码大小

      • more code = more downloading = …

        更多代码 = 更多下载 = ...

JAVA

Apache Struts 2

Free open source framework for creating enterprise-ready Java-based web applications

用于创建企业就绪的基于 Java 的 Web 应用程序的免费开源框架

Action-based MVC Model 2 (Pull MVC) framework combining Java Servlets and JSP technology

结合Java Servlets和JSP技术的基于动作的MVC模型2（Pull MVC）框架

  • model

    • action (basic building blocks) from which the view can pull information via the ValueStack

    • 视图可以通过 ValueStack 从中提取信息的操作（基本构建块）

    • action represented by POJO (Plain Old Java Object) following the JavaBean paradigm and optional helper classes

      由POJO (Plain Old Java Object)表示的操作，遵循 JavaBean 范式和可选的帮助器类

  • view

    • template-based approach often based on JavaServer Pages (JSP) in combination with tag libraries (collection of custom tags)

      基于模板的方法通常基于JavaServer Pages (JSP)结合标签库（自定义标签的集合）

  • controller

    • based on Java Servlet filter in combination with interceptors

      基于Java Servlet过滤器结合拦截器interceptors

MVC Model 2 (MVC 2) in Struts 2

Apache Struts 2 Architecture

  1. Receive request 接收请求

    • filter chain 过滤器链

      • interception of requests and responses, e.g. XSLT transformation

        请求和响应的拦截，例如 XSLT 转换

  2. Execute relevant Action 执行相关操作

    • invoke interceptors

      调用拦截器

    • read/update data (model)

      读取/更新数据

  3. Build response 构建响应 (view)

    • often based on JSP template

      通常基于 JSP 模板

    • interceptors in reverse order

      反序拦截器

  4. Send response 发送回复

    • again through filter chain

      再次通过过滤器链

Spring Framework
  • Java application framework
  • Various extensions for web applications
  • Modules

    • data access 数据存取
    • inversion of control container 控制容器反转
    • convention-over-configuration 约定优于配置
    • remote access framework 远程访问框架
    • transaction management 交易管理
    • authentication and authorisation 认证和授权
    • …

PHP

Yii Framework

PHP framework for the development of Web 2.0 applications that offers a rich set of features

  • AJAX-enabled widgets (Asynchronous JavaScript and XML)

    支持 AJAX 的小部件（异步 JavaScript 和 XML）

  • web service integration

    网络服务集成

  • authentication and authorisation

    认证和授权

  • flexible presentation via skins and themes

    通过皮肤和主题灵活呈现

  • Data Access Objects (DAO) interface to transparently access different database management systems

    数据访问对象 (DAO) 接口以透明地访问不同的数据库管理系统

  • integration with the jQuery JavaScript library

    与 jQuery JavaScript 库的集成

  • layered caching

    分层缓存

  • ...

Zend

Open source PHP framework offering various features

  • MVC architectural pattern
  • loosely coupled components
  • object orientation
  • flexible caching
  • simple Cloud API
  • features to deal with emails (POP3, IMAP4, …)
  • …
CakePHP

Open source PHP web application framework

  • MVC architectural pattern
  • rapid prototyping via scaffolding
  • authentication
  • localisation

  • caching
  • validation
  • …

Python

Django

Open source Python web application framework

  • MVC architectural pattern
  • Don’t Repeat Yourself (DRY)
  • object-relational mapper
    • mapping between model (Python classes) and a relational database
  • integrated lightweight web server
  • localisation
  • caching
  • ...
Flask

Open source Python microframework

  • MVC architectural pattern
  • built-in development server and debugger
  • integrated unit testing support
  • RESTful request dispatching
  • templating through Jinja2
  • friendly Web Server Gateway Interface (WSGI) via Werkzeug
  • support for secure cookies (client side sessions)
  • ...

Ruby

Ruby on Rails (RoR)
  • Open source web application framework
  • Combination of
    • dynamic, reflective, object-oriented programming language Ruby
      • combination of Perl-inspired syntax with "Smalltalk features"
    • web application framework Rails
  • Based on MVC architectural pattern

    • structure of a webpage separated from its functionality via the unobtrusive JavaScript technique

      通过不显眼的 JavaScript技术将网页的结构与其功能分离

  • The scaffolding feature offered by Rails can automatically generate some of the models and views that are required for a website
    • developer has to run an external command to generate the code
  • Ruby on Rails Philosophy
    • Don't Repeat Yourself (DRY)
      • information should not be stored redundantly (e.g. do not store information in configuration files if the data can be automatically derived by the system)
    • Convention over Configuration (CoC)
      • programmer only has to specify unconventional application settings
      • naming conventions to automatically map classes to database tables (e.g. by default a 'Sale' model class is mapped to the 'sales' database table)
  • High modularity
  • Plug-ins can be added for desired server-side functionality
  • Use RubyGems package manager to easily add plug-ins (“gems”)

JavaScript

Node.js

  • Server-side JavaScript

    • low-level, comparable to functionality offered by Servlets

      低级，可与 Servlet 提供的功能相媲美

    • handling post/get requests, database, sessions, …

      处理 post/get 请求、数据库、会话等

  • Write your entire app in one language

    • however, server-side and client-side code still separated

      然而，服务器端和客户端代码仍然分开

    • SSR (Server Side Rendering) of client-side HTML is possible

      客户端 HTML 的 SSR（服务器端渲染）是可能的

  • Built-in web server (no need for Apache, Tomcat, etc.)

    内置网络服务器（不需要Apache、Tomcat等）

  • High modularity

    • packages can be added for additional functionality (via npm)
    • many available frameworks (Express, Passport, Sequelize,…)
    • HTTP utility methods (routing, sessions, ...)
    • template engines (Jade, EJS, …)
Node.js Packages
  • Express, EJS: web server and template engine
    • customizable web server
    • render templates similar to JSP
  • Passport: session + authentication
    • basic login, API login using Authorization header
    • social media login (Google, Facebook, …)
  • Sequelize: ORM database framework
    • do not worry about SQL queries
    • do not worry about basic security issues
  • Many more + combinations
    • Combine PDFPhantom + EJS and you can create PDF documents from EJS templates (invoices, …)
Deno

Server-side JavaScript

  • Created by Node.js founder
  • Improved on ‘issues’ in Node.js
    • Secure runtime (i.e. explicit permission for network and file access)
    • Standard TypeScript support
    • ES6 imports instead of npm packages

.NET

ASP.NET MVC

Web framework for .NET languages (C\#, VB)

  • MVC architectural pattern

  • Inversion of control container

  • Extensible and pluggable framework

    可扩展和可插拔的框架

    • can use any existing .NET libraries in the form of DLL files

      可以以 DLL 文件的形式使用任何现有的.NET 库

    • core components such as view engine or URL routing modules can be replaced

      核心组件，如视图引擎或 URL 路由模块可以替换

  • Can generate some of the client-side JavaScript for you

    • form validation
    • dynamic updates using AJAX

  • Localisation

  • Session management

  • Caching

  • …

Client-side web frameworks 客户端框架

React

  • Users can define rich reusable UI components

    • small bundles of HTML, CSS and JS related to a UI component
    • JSX files (Combination of JavaScript and HTML)

  • React Native → React with native iOS or Android components instead of HTML DOM

    React Native → 使用原生 iOS 或 Android 组件而不是 HTML DOM 进行反应

AngularJS (LTS)

  • Two-way data binding between models and views

    模型和视图之间的双向数据绑定

  • The new Angular (Typescript) is preferred for new projects

  • Users can define rich reusable UI components

Angular

  • Two-way data binding between models and views

  • Users can define rich reusable UI components

    用户可以定义丰富的可重用 UI 组件

    • small bundles of HTML, CSS and JS related to a UI component

@Component({
    selector: 'my-images',
    providers : [ImageService],
    styles: [ ` div { background-color: blue;} `]
    template: `
		<div class='image_gallery'>
			<div *ngFor="#i of images">
				<img src="{{i.path}}" alt="i.description">
			</div> 
		</div> `
})

Vue.js

  • Combination of React + Angular
    • React + Angular: Component-based
    • React + Angular: Animations and transitions
    • React + Angular: SSR (Server-side rendering)
    • React: JSX support + render functions
    • Angular: Directives
    • Angular: Two-way data binding
    • Angular: Watchers
  • Two-way data binding between models and views
  • Users can define rich reusable UI components
  • Many companies are using Vue
    • Adobe, NASA, GitLab, Nintendo, …

Ember.js

  • Built for single-page web applications

    专为单页 Web 应用程序构建

    • information is refreshed dynamically on demand

      信息按需动态刷新

    • no page refreshes required

      无需页面刷新

  • Comes with Ember Data, a data persistence library

    附带 Ember Data，一个数据持久化库

    • provides facilities for object-relational mapping (ORM)

      为对象关系映射 (ORM) 提供便利

    • maps client-side models to server-side data

      将客户端模型映射到服务器端数据

    • usage is optional, you can also provide your own interface to the server-side data

      用法是可选的，您也可以提供自己的服务器端数据接口

  • Users can create custom HTML tags ("components")

    用户可以创建自定义 HTML 标签（“组件”）

    • can also include logic relevant to the new tag’s function

      还可以包括与新标签功能相关的逻辑

      • e.g. handle user input or deal with child elements placed in between the custom tag

        例如处理用户输入或处理放置在自定义标签之间的子元素

Creating desktop applications

NW.js

  • Formerly node-webkit project

  • Bundles your app for usage as desktop application

    • Webkit + Node.js + Application Files (HTML, JS, CSS, …)

  • Lets you call Node.js modules directly from DOM

    直接从 DOM 调用 Node.js 模块

    • e.g. use filesystem module to read and write files

      例如使用文件系统模块来读写文件

  • Included API provides access to native UI features

    • right-click context menu
    • system tray icon
    • notifications
    • …

Electron

  • Similar to NW.js (similar native APIs)
  • Developed for GitHub's Atom editor, now widely used

Meteor

  • Full stack framework (server and client side)

    全栈框架（服务器端和客户端）

  • Integrates with many existing technologies

    • MongoDB
    • Vue, React, Angular
    • NPM (Node.js), Cordova

  • Large focus on getting data from server to client and client to server without creating your own API endpoints

    专注于从服务器到客户端以及从客户端到服务器获取数据，而无需创建自己的 API 端点

  • Example: Used on BigBlueButton

Web Content Management Systems 网页内容管理系统

Content management systems that focus on web content

专注于网络内容的内容管理系统

  • Main functionality 主要功能

    • data storage and publishing 数据存储和发布
    • user management 用户管理
      • including access rights 访问权限
    • versioning 版本控制
    • workflows 工作流

  • Offline (create static webpages), online (create webpages on the fly) and hybrid systems

    离线（创建静态网页）、在线（动态创建网页）和混合系统

  • Graphical interface for creating and managing content

    用于创建和管理内容的图形界面

  • Suited for non-technical users since the underlying technology is normally completely hidden

    适合非技术用户，因为底层技术通常是完全隐藏的

  • Web CMS Examples

    • Joomla, Drupal, WordPress, ...
