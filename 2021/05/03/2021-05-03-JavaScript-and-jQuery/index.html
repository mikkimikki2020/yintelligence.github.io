JavaScript教程

JavaScript

  • High-level loosely typed dynamic language

    高级松散类型动态语言

  • Introduced by Netscape in 1995

    • lightweight interpreted client-side programming

      轻量级解释型客户端编程

  • Supports imperative, object-oriented and functional programming styles

    支持命令式、面向对象和函数式编程风格

  • Nowadays also used in non-web-based environments

    现在也用于非基于 Web 的环境中

    • PDF, Microsoft Office, …

  • JavaScript implements the ECMAScript specification

    JavaScript 实现了 ECMAScript 规范

    • latest version is ECMAScript 2019

Adding JavaScript to a Webpage 添加到网页

<!DOCTYPE html>
<html>
    <head>
    <tile>JavaScript Example</tile>
        ...
        <script>
            alert("Message in the header");
        </script>
        <script src="example.js"></script>
    </head>
    <body>
        <h1>A First Example</h1>
        <script>
            document.write("<p>Text added by JavaScript</p>");
        </script>
        <p id="p2">A second paragraph.</p>
    </body>
</html>

  • JavaScript code can be placed in the <head>, in the <body> or in external files

    JavaScript代码可以被放置在<head>，在<body>或在外部文件

  • Advantages of external JavaScript files

    外部 JavaScript 文件的 优点

    • separation of HTML and code

      HTML 和代码的分离

    • faster page load if JavaScript file is cached

      如果 JavaScript 文件被缓存，页面加载速度会更快

  • JavaScript code placed just before the </body> tag (end of the page) might improve page load

    放置在</body>标记之前（页面末尾）的JavaScript 代码可能会改善页面加载

    • display of HTML is not blocked by script loading

      脚本加载不会阻止 HTML 的显示

    • alternative: attribute defer="true" in the <script> tag

      替代：属性defer="true"在<script>标签

  • JavaScript errors silently ignored by most browsers

    大多数浏览器会默默忽略 JavaScript 错误

    • activate browser console for debugging (e.g. F12)

      激活浏览器控制台进行调试（例如 F12）

    • Error aalert shows up in the console

      错误aalert显示在控制台中

  • Use the latest version of the language by adding "use strict"; to the top of a script

    通过添加“ use strict”来使用最新版本的语言；到脚本的顶部

Output 输出

There are different possibilities for JavaScript output

JavaScript 输出有不同的可能性

  • document.write()

    • adds content to the page while loading

      加载时向页面添加内容

    • deletes (overwrites) page if used after page has been loaded

      如果在页面加载后使用，则删除（覆盖）页面

    • should only be used for testing (slow)

      如果在页面加载后使用，则删除（覆盖）页面

  • console.log()

    • outputs content to the browser console

      将内容输出到浏览器控制台

  • window.alert() or alert()

    • outputs data in an alert message box

      在警报消息框中输出数据

  • innerHTML property

    • adds content to an HTML element (via DOM tree)

      向 HTML 元素添加内容（通过 DOM 树）

document.getElementById("p2").innerHTML = "A new second paragraph.";

Variables and Data Types 变量和数据类型

  • Variables have dynamic types 变量具有动态类型

    let x; // x has been declared but the value is 'undefined'
x = "Albert Einstein"; // now x is a string
x = 3.1415; // now x is a number
typeof x; // number
x = false; // now x is a boolea

  • Primitive data types 变量具有动态类型

    • string, number and boolean
    • null and undefined

  • Objects

Strings

  • Any text between single (') or double quotes (")

  • Strings are immutable

    字符串是不可变的

  • Special characters (e.g. \, \t or \n) have to be escaped with a backslash (\)

let hello = "Hello World";
let correct = "This is \"correct\""; // escaped double quotes
let tmp = hello.length; // 11
tmp = hello + ". " + correct; // Hello World. This is "correct"
tmp = hello.charAt(1); // e
tmp = hello.toUpperCase(); // HELLO WORLD
tmp = hello.replace("World", "VUB"); // replace first occurence of 'World' with 'VUB'

Numbers

  • Only one type of numbers (64-bit floating point)

    • mantissa (52 bits), exponent (11 bits) and sign (1 bit)

      尾数（52 位）、指数（11 位）和符号（1 位）

    let x = 42.00;
let y = 42;
let z = 42e3; // 42000

  • NaN indicates that a value is not a number

    NaN表示值不是数字

    let x = 42 / "foo"; // but what about 'x = 42 + "foo"?
isNaN(x); // returns true
// what is the difference between '10 + 10 + "foo"' and '"foo" + 10 + 10'?

  • Infinity for values that are too large to be represented

    let x = 42 / 0; // x becomes Infinity

  • Rounding errors 舍入误差

    • due to rounding errors we might not always get the expected result

      由于舍入错误，我们可能并不总能得到预期的结果

    let x = 0.3 + 0.6; // 0.8999999999999999
let x = (0.3 * 10 + 0.6 * 10) / 10; // possible solution (resulting in 0.9)

    Boolean

Booleans can have the values true or false

let x = true;
let y = false;
let z = 5 > 3; // true

Date and Math Objects

  • Date object can be used to work with dates

    日期对象可用于处理日期

    let now = new Date();
let nextWeek = new Date("2017-11-10T14:00:00"); // ISO 8610 syntax
let milis = now.getTime(); // time in milliseconds since January 1, 1970
let test = nextWeek.getTime() > now; // true

  • Math object can be used for various mathematical operations and offers some constans (e.g. PI and E)

    Math 对象可用于各种数学运算并提供一些常量（例如 PI 和 E）

    Math.random(); // random number between 0 (inclusive) and 1 (exclusive)
Math.round(3.6); // 4 (rounded to the nearest integer)
Math.max(4,10,3); // 10 (number with the highest value)

    Objects

  • Container of properties (name/value) where a value can be any JavaScript value (including other objects)

    属性（名称/值）的容器，其中值可以是任何 JavaScript 值（包括其他对象）

    let student = {
    firstName: "John",
    lastName: "Harper",
    height: 182
};

  • Two possibilities to retrieve object values

    检索对象值的两种可能性

    • second option only works for legal JavaScript names and non-reserved words

      第二个选项仅适用于合法的 JavaScript 名称和非保留字

    student["firstName"];
student.lastName;

  • Existing values can be updated, new properties can be added and existing properties can be deleted

    可以更新现有值，可以添加新属性，可以删除现有属性

    student.lastName = "Wayne"; 
student.address = {
	street: "Pleinlaan",
	number: 2
};
delete student.lastName;

  • Objects are passed by reference and not by value

    对象通过引用而不是通过值传递

    let x = student; 				// student object created before
x.firstName = "Peter";
let name = student.firstName; 	// name is "Peter" since x and student are references
								// to the same object

  • Cloning of objects (deep copy) is not trivial in JavaScript

    对象的克隆（深拷贝）在 JavaScript 中并不简单

    • e.g. jQuery offers a clone() method

Arrays

  • An array is a variable that can hold multiple values which can be accessed via an integer offset

    数组是一个可以保存多个值的变量，这些值可以通过整数偏移量访问

    • values of different types might be mixed

      不同类型的值可能会混合

    let points = [10, 5, 17, 42];
points.length; // number of elements (4)
points[1]; // accesses the element with the given offset (5)
points[points.length] = 13; // adds 13 to the end of the array [10, 5, 17, 42, 13]
let last = points.pop(); // removes the last element [10, 5 , 17 , 42]
let l = points.push(2); // adds 2 at the end and returns the length (5)
let first = points.shift(); // removes the first element [5, 17, 42, 2]
l = points.unshift(7); // adds 7 to the beginning and returns the length (5)
delete points[2]; // the third element will be undefined
let matrix = [[3, 1, 2], [9, 6, 4], [5, 7, 8]]; // array of arrays ("2D" array)
matrix [1][2]; // 4

  • Other methods to sort arrays, combine arrays etc.

Functions 函数

A function is a block of code to perform a particular task

  • enables the reuse of code

  • functions are objects and can be used like any other value

    函数是对象，可以像任何其他值一样使用

    • can be stored in variables, objects and arrays
    • can be passed as argument to functions or returned from functions

  • access to a function without the () operator returns the definition of a function

function multiply(p1, p2) {
	return p1 * p2;
}
let r = multiply(3, 4); // 1

Scope 范围

  • Local variables declared within a function have block-level scoping

    在函数中声明的局部变量具有块级作用域

  • A variable declared outside of a function has global scope

    在函数外声明的变量具有全局作用域

  • Local variables deleted when the block is completed

    块完成时删除局部变量

  • Function arguments (parameters) work as local variables within functions

    函数参数（参数）作为函数内的局部变量

  • Warning

    • old var syntax does not offer block-level scoping but rather provides function scoping

      旧var语法不提供块级范围，而是提供函数范围

Objects Revisited 重访对象

  • Functions can be added to objects (methods)

    函数可以添加到对象（方法）

    // Let us assume that we have the student object that was created earlier
	student.fullName = function() {
return this.firstName + " " + this.lastName;
}

  • Multiple objects can be created via an object constructor function (prototype)

    可以通过对象构造函数（原型）创建多个对象

    function student(firstName, lastName, height) = {
    this.firstName = firstName;
    this.lastName = lastName;
    this.height = height;
    this.fullName = function() {return this.firstName + " " + this.lastName;}
};

let student1 = new student("SpongeBob", "SquarePants", 174);
let student2 = new student("Patrick", "Star", 177);

    Classes 类

Classes have been introduced in ECMAScript 2015

  • simplifies inheritance etc.
class Person {
    constructor(firstName, lastName, height) {
        this.firstName = firstName;
        this.lastName = lastName;
        this.height = height;
    }
    
    fullName() {
        return this.firstName + " " + this.lastName;
    }
}

class Student extends Person {
    constructor(firstName, lastName, height, studentID) {
        super(firstName, lastName, height);
        this.studentID = studentID;
    }
}

let student = new Student("Audrey", "Sanctorum", 174, 93283982);
console.log(student.fullname()); // Audrey Sanctorum
console.log(student instanceof Person); // true
console.log(student instanceof Student); // true

Modules 模块

Modules can be used to control the scoping instead of using a shared-everything approach

模块可用于控制范围，而不是使用共享一切的方法

  • export keyword to expose code to other modules

    export 将代码暴露给其他模块的关键字

  • exported functionality can be accessed from another module via the import keyword

    可以通过import关键字从另一个模块访问导出的功能

export let counter = 0;
export function multiply(p1, p2) {
	return p1 * p2;
}
...
import multiply from "./example.js";
...

Loops

For Loop

for (let i = 0; i < 10; i++) {
    text += "Number: " + i + "<br />";
}

For/In Loop

  • iterate over object properties
for (x in student) {
    text += " " + student[x]; // Add all properties of 'student' to text string

While Loop

while (i < 10) {
    text += "Number: " + i + "<br />";
    i++;

Events 事件

  • Various events can be triggered when accessing an HTML document

    访问 HTML 文档时可以触发各种事件

    • document has finished loading 文件已完成加载
    • form field has been changed 表单字段已更改
    • button has been clicked 按钮已被点击
    • mouse is moved over an image 鼠标在图像上移动
    • ...

  • JavaScript can be used to trigger some actions when events are detected

    JavaScript 可用于在检测到事件时触发某些操作

<button onclick="startSlideshow()">Start</button>

JavaScript Best Practices

  • Avoid (minimise) global variables 避免（最小化）全局变量

    • global variables can lead to name collisions for sub-programs but modules can be used to fix the problem

      全局变量可能会导致子程序的名称冲突，但可以使用模块来解决问题

  • Always initialise variables when they are declared

    声明变量时总是初始化变量

    • avoids undefined values 避免未定义的值

  • Always use the === and !== equality operators

    • == and != do some automatic type coercion (with complicated rules) if the types are not equal

  • Avoid block-less if, while, do or for statements

    if (x > 10) {
	t = true;
}

  • Avoid the use of eval()

    • runs text as code and leads to performance and security issues

      将文本作为代码运行并导致性能和安全问题

jQuery

  • JavaScript library that simplifies the use of JavaScript on websites

    简化 JavaScript 在网站上使用的 JavaScript 库

    • HTML/DOM traversal and manipulation

      HTML/DOM 遍历和操作

    • event handling 事件处理

    • effects and animation 效果和动画

    • AJAX

  • Most popular and extendable JavaScript framework

    最流行和可扩展的 JavaScript 框架

    • large developer community 大型开发者社区

  • Two ways to add jQuery to a website

    将 jQuery 添加到网站的两种方法

    1. download jQuery library (JavaScript file) from jquery.com

      从 jquery.com下载 jQuery 库（JavaScript 文件）

    2. link to jQuery library on content delivery network (CDN) server

    链接到内容交付网络 (CDN) 服务器上的 jQuery 库

    • Microsoft, Google, …

  • jQuery code executed when document loaded (ready)

    加载文档时执行的 jQuery 代码（就绪）

<!DOCTYPE html>
<html>
    <head>
        <tile>jQuery Example</tile>
        ...
        <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js">
        </script>
        <script>
            $(document).ready(function(){
                // jQuery code
            });
        </script>
    </head>
    <body>
        ...
    </body>
</html>

DOM Revisited

  • cross-browser issues when accessing DOM via JavaScript

    通过 JavaScript 访问 DOM 时的跨浏览器问题

jQuery Syntax 语法

jQuery syntax makes it easy to select HTML elements (based on CSS-style selectors) and perform some actions on these elements

jQuery 语法可以轻松选择 HTML 元素（基于 CSS 样式选择器）并对这些元素执行一些操作

  • $ sign to access jQuery followed by a selector and the action to be performed on the selected elements

    $访问 jQuery 的符号，然后是选择器和要对所选元素执行的操作

$(selector).action();

Example:

$("h1").hide(); // hide all h1 elements (implicit iteration)
$("h1").hide().delay(500).fadeIn(1500); // multiple methods can be chained
$("h1:first").hide(); // hides the first h1 element
$("#main").hide(); // hides the element with id=main
$(".note").toggle(); // toggles all elements of class note
$("p:even").toggle(); // toggles all even paragraph elements
$("h1").html(); // get content from first h1 element
$("h1").each(function() {...}); // loop over each h1 element
$("p").html("New"); // updates the content of all p elements to "New"

Caching jQuery Selections 缓存 jQuery 选择

A jQuery object (result of a selection) has references to the elements in the DOM tree

一个 jQuery 对象（选择的结果）引用了 DOM 树中的元素

  • if the same selection is used multiple times, the jQuery object can be reused (cached) by storing it in a variable

    如果多次使用相同的选择，则可以通过将 jQuery 对象存储在变量中来重用（缓存）它

let $paragraphs = $("p"); // often prefix $ used for variables with a jQuery object

Attributes 属性

  • Attributes can be accessed and updated

    可以访问和更新属性

    $("a:first").attr("href"); // get the href attribute of the first anchor tag
$("a:first").attr("href", "http://wise.vub.ac.be"); // update attribute
$("a:first").removeAttr("href"); // remove href attribute of the first anchor tag

  • Similar functionality exists for classes

    $("p").addClass("main"); // adds an additional class
$("p").removeClass("main"); // removes the class main

    Event Handling 事件处理

  • Deals with cross-browser issues behind the scenes

    在幕后处理跨浏览器问题

  • There are various jQuery events from different sources

    有来自不同来源的各种 jQuery 事件

    • keyboard
      • input, keydown, keyup, keypress
    • mouse
      • click, dblclick, mouseup, mousedown, mouseover, mousemove, mouseout, hover
    • UI
      • focus, blur, change
    • form
      • submit, select, change
    • document
      • ready, load, unload

  • We can handle events via the on() method

    我们可以通过on()方法处理事件

    1. the first parameter is the event to respond to

      第一个参数是要响应的事件

    2. the second parameter is a named or anonymous function

      第二个参数是命名或匿名函数

$("#start").on("click", startSlideshow();); // calls a named function
$("p").on("mouseover", function(e) { // anonymous function
	$(this).hide();
});

DOM Navigation

Various methods can be used to start navigating the DOM tree from a given selection

可以使用各种方法从给定的选择开始导航 DOM 树

  • parent()
    • direct parent of current selection
  • parents()
    • all parents of current selection
  • `children()
    • all children of current selection
  • next ()
    • next sibling of current selection
  • siblings()
    • all siblings of current selection
  • ...

JavaScript Object Notation (JSON)

  • Developed as an XML alternative to represent JavaScript objects as strings (language independent)

    作为 XML 替代方案开发，将 JavaScript 对象表示为字符串（独立语言）

  • Easy to produce and faster to parse than XML

    比 XML 易于生成且解析速度更快

    • supports different data types

      支持不同的数据类型

  • JSON is based on a subset of JavaScript

    JSON 基于 JavaScript 的一个子集

    • JSON document can be read via the JavaScript eval() function

      JSON 文档可以通过 JavaScript eval() 函数读取

      • security issues: note that this approach can be dangerous if the source is not trusted since any JavaScript code might be executed

        安全问题：请注意，如果来源不受信任，这种方法可能很危险，因为可能会执行任何 JavaScript 代码

    • most recent browsers offer a JSON parser

      最近的浏览器提供了一个 JSON 解析器

      • recognises only JSON data types and rejects any scripts

        仅识别 JSON 数据类型并拒绝任何脚本

  • Many Web 2.0 Applications offer a JSON interface

    许多 Web 2.0 应用程序提供 JSON 接口

    • Flickr, YouTube, Delicious, ...

JSON Data Types 数据类型

Number integer, real or float

String double-quoted Unicode (use backslash for escaping)
Array comma-separated ordered sequence of values enclosed in square brackets
Object comma-separated collection of key:value pairs enclosed in curly brackets
null null value
